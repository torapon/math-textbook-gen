# ロルの定理と平均値の定理

## なぜ重要か

導関数は *各点ごと* に定義されますが、私たちはそれを使って絶えず *大域的* な結論を
導きます — 「$`f' > 0`$ だから $`f`$ は増加する」「$`f' = 0`$ だから $`f`$ は定数」。
局所的な傾きから大域的な振る舞いへの架け橋が **平均値の定理**（MVT）です。下流のすべて
— 単調性の判定、誤差評価、Taylor の定理 — はこの橋を渡ります。橋は3段階で築きます —
内部極値の補題、ロルの定理、そして MVT です。

## フェルマーの内部極値補題

> **補題（フェルマー）.** $`f`$ が *内部* の点 $`c`$ で極大または極小をとり、そこで微分
> 可能ならば、$`f'(c) = 0`$ である。

*証明.* $`c`$ を極大とすると、小さい $`h`$ すべてで $`f(c + h) \le f(c)`$。$`h > 0`$ では
差分商 $`\dfrac{f(c+h) - f(c)}{h} \le 0`$ なので、極限をとって $`f'(c) \le 0`$。$`h < 0`$
では商は $`\ge 0`$（非正の分子を負の分母で割る）なので $`f'(c) \ge 0`$。両方が成り立つので
$`f'(c) = 0`$。（極小は不等号を逆にした同じ議論です。）$`\blacksquare`$

幾何的には、内部の山や谷では接線が水平です。*内部* という語が本質的です — 端点では片側
しか存在せず、傾きは消えるとは限りません。

## ロルの定理

> **定理（ロル）.** $`f`$ が $`[a, b]`$ で連続、$`(a, b)`$ で微分可能、かつ
> $`f(a) = f(b)`$ ならば、$`f'(c) = 0`$ となる点 $`c \in (a, b)`$ が存在する。

*証明.* [最大値・最小値の定理](./10-extreme-value-theorem.ja.md)により、$`f`$ は
$`[a, b]`$ で最大値と最小値を達成します。**両方** が端点で達成されるなら — $`f(a) = f(b)`$
なので — 最大値と最小値が等しく、$`f`$ は定数で、すべての $`c \in (a,b)`$ で $`f'(c) = 0`$。
そうでなければ、ある極値が内部の点 $`c \in (a, b)`$ で達成されます。そこで $`f`$ は微分
可能なので、フェルマーの補題から $`f'(c) = 0`$。$`\blacksquare`$

## 平均値の定理

ロルの定理は、弦が水平な特別な場合です。それを傾ければ一般の主張が得られます — どこかで
接線が割線に平行になるのです。

> **定理（平均値の定理）.** $`f`$ が $`[a, b]`$ で連続、$`(a, b)`$ で微分可能ならば、
> ```math
> f'(c) = \frac{f(b) - f(a)}{b - a}
> ```
> となる点 $`c \in (a, b)`$ が存在する。

*証明.* 割線を引き去ります。
```math
h(x) = f(x) - f(a) - \frac{f(b) - f(a)}{b - a}\,(x - a)
```
と定義すると、$`h`$ は $`[a,b]`$ で連続、$`(a,b)`$ で微分可能です。すぐに確かめられるように
$`h(a) = 0`$、$`h(b) = f(b) - f(a) - (f(b) - f(a)) = 0`$、よって $`h(a) = h(b)`$。ロルの
定理により $`h'(c) = 0`$ となる $`c \in (a,b)`$ が存在し、すなわち
```math
0 = h'(c) = f'(c) - \frac{f(b) - f(a)}{b - a}. \qquad \blacksquare
```

量 $`\dfrac{f(b)-f(a)}{b-a}`$ は $`[a,b]`$ 上の *平均* 変化率です。MVT は、この平均が
ある内部の点で *瞬間* 変化率として達成される、と述べています。

## 帰結：$`f'`$ から $`f`$ へ戻る

これらの系こそ MVT が重要な理由です — $`f'`$ の符号の情報を、$`f`$ の形の情報へと変換
します。以下、$`f`$ は区間上で微分可能とします。

- **導関数が $`0`$ $`\Rightarrow`$ 定数.** $`f' = 0`$ が至る所なら、任意の $`x < y`$ に
  対し MVT が $`f(y) - f(x) = f'(c)(y - x) = 0`$ を与え、$`f`$ は定数です。
- **導関数が等しい $`\Rightarrow`$ 定数差.** $`f' = g'`$ なら、上を $`f - g`$ に適用
  します。（これが原始関数の一意性の背後にあります。）
- **$`f'`$ の符号が単調性を支配する.** 区間で $`f' > 0`$ なら、$`x < y`$ で
  $`f(y) - f(x) = f'(c)(y - x) > 0`$ なので $`f`$ は狭義単調増加。同様に $`f' < 0`$ で
  狭義単調減少、$`f' \ge 0`$ で単調非減少です。

## 例題

**問題.** すべての実数 $`x, y`$ で $`|\sin x - \sin y| \le |x - y|`$ を示しなさい。

端点 $`x`$, $`y`$ の区間で $`\sin`$ に MVT を適用すると、その間のある $`c`$ で
```math
\sin x - \sin y = \cos(c)\,(x - y).
```
$`|\cos c| \le 1`$ なので、絶対値をとって $`|\sin x - \sin y| = |\cos c|\,|x - y| \le
|x - y|`$。

**答え.** 不等式は成立します — $`\sin`$ は定数 $`1`$ で *リプシッツ* です。（同じ枠組みが、
任意の評価 $`|f'| \le M`$ を $`|f(x) - f(y)| \le M|x - y|`$ に変えます。）

## よくある間違い

- **仮定を1つ落とす.** 弦の例 $`f(x) = |x|`$（$`[-1, 1]`$ 上）は $`f(-1) = f(1)`$ ですが、
  $`f'`$ の内部の零点は *存在しません* — $`f`$ が $`0`$ で微分不可能だからです。ロルと MVT
  は開区間 *全体* での微分可能性を必要とします。
- **一意な、あるいは位置のわかる $`c`$ を期待する.** これらの定理は *存在* を主張します。
  そのような $`c`$ は複数あり得るし、公式も与えません。
- **2つの区間を混同する.** 連続性は *閉* 区間 $`[a,b]`$（端点を含む）で、微分可能性は
  *開* 区間 $`(a,b)`$ でのみ必要です。
- **結論を逆に読む.** $`f' > 0`$ は増加を与えますが、「増加」は至る所 $`f' > 0`$ を与え
  ません（$`f(x) = x^3`$ は増加しますが $`f'(0) = 0`$）。

## 練習問題

1. $`[0, 2]`$ 上の $`f(x) = x^2 - 2x`$ でロルの定理を確かめ、点 $`c`$ を求めなさい。
2. MVT を用いて、すべての $`x > 0`$ で $`\ln(1 + x) < x`$ を証明しなさい。
3. 区間のすべての $`x`$ で $`f'(x) = 0`$ ならば、そこで $`f`$ が定数であることを証明
   しなさい。
4. $`|\arctan a - \arctan b| \le |a - b|`$ を示しなさい。

<details>
<summary>解答</summary>

1. $`f(0) = 0 = f(2)`$ で、$`f'(x) = 2x - 2 = 0`$ は $`c = 1 \in (0,2)`$。
2. $`g(t) = \ln(1+t)`$ に $`[0, x]`$ で MVT を適用：ある $`c \in (0, x)`$ で
   $`\ln(1+x) - 0 = \dfrac{1}{1+c}\,x`$。$`c > 0`$ なので $`\dfrac{1}{1+c} < 1`$、ゆえに
   $`\ln(1+x) < x`$。
3. 区間内の任意の $`x < y`$ で MVT が $`f(y) - f(x) = f'(c)(y - x) = 0`$ を与え、
   $`f(y) = f(x)`$。その共通の値が定数です。
4. MVT により、ある $`c`$ で $`\arctan a - \arctan b = \dfrac{1}{1 + c^2}(a - b)`$。
   $`\dfrac{1}{1+c^2} \le 1`$ なので、絶対値をとって $`\le |a - b|`$。

</details>
