# 区間縮小法

## なぜ重要か

解析学の多くの構成では、互いに近づいていく2つの境界の間に数を挟み込んで捕らえます。
二分法、実数の2進展開、そして
[中間値の定理](./09-intermediate-value-theorem.ja.md)の証明は、いずれもこの仕組みです。
区間縮小法は、このような挟み撃ちが — 長さが $`0`$ に縮むかぎり — 必ず **ちょうど1つ** の
実数を捕らえることを保証します。これも完備性のもう1つの顔であり、最も幾何学的な
ものです。

## 定義

> **定義（区間縮小列）.** 閉区間の列 $`I_n = [a_n, b_n]`$ が *縮小列（入れ子）* で
> あるとは、
> $$
> I_1 \supseteq I_2 \supseteq I_3 \supseteq \cdots,
> \quad\text{すなわち}\quad a_n \le a_{n+1} \le b_{n+1} \le b_n \ \text{ (すべての } n)
> $$
> が成り立つことをいいます。

各区間は1つ前の区間の内側にあります。長さが *$`0`$ に縮む* とは $`b_n - a_n \to 0`$ の
ことです。

## 主な結果

> **定理（区間縮小法）.** $`I_n = [a_n, b_n]`$ を入れ子の **閉** 区間とする。このとき
> 共通部分は空でない：
> $$
> \bigcap_{n=1}^{\infty} I_n \neq \varnothing.
> $$
> さらに $`b_n - a_n \to 0`$ ならば、共通部分は **ただ1点** である：
> $`\{\xi\} = \bigcap_n I_n`$ となる $`\xi`$ がただ1つ存在し、$`a_n \to \xi`$,
> $`b_n \to \xi`$ となる。

*証明.* 左端 $`(a_n)`$ は（入れ子により）増加し、$`b_1`$ で上に有界です
（$`a_n \le b_n \le b_1`$）。[単調収束定理](./04-monotone-convergence.ja.md)により
$`\xi = \sup_n a_n`$ に収束します。対称的に $`(b_n)`$ は $`\eta = \inf_n b_n`$ に減少し、
すべての $`n`$ で $`a_n \le \xi \le \eta \le b_n`$ なので $`[\xi, \eta] \subseteq I_n`$、
ゆえに $`[\xi,\eta] \subseteq \bigcap_n I_n`$ となり共通部分は空でありません。

さらに $`b_n - a_n \to 0`$ ならば $`0 \le \eta - \xi \le b_n - a_n \to 0`$ より
$`\eta = \xi`$。共通部分の任意の点はすべての $`n`$ で $`[a_n, b_n]`$ に属するので $`\xi`$ に
等しく、共通部分はちょうど $`\{\xi\}`$ です。$`\blacksquare`$

## 外せない2つの仮定

- **閉.** 開区間では成り立ちません：$`J_n = \left(0, \tfrac1n\right)`$ は入れ子ですが
  $`\bigcap_n J_n = \varnothing`$ — 極限点になるはずの $`0`$ がどの区間からも除かれて
  います。
- **有界.** 非有界区間では成り立ちません：$`K_n = [n, \infty)`$ は入れ子ですが
  $`\bigcap_n K_n = \varnothing`$。（定理は有限閉区間 $`[a_n,b_n]`$ について述べています。）

$`\mathbb{Q}`$ 上でも成り立ちません。$`\sqrt2`$ に近づく有理端点の区間は、*$`\mathbb{Q}`$ の
中では* 共通部分が空です。よってこの定理も、本シリーズの他の定理と同様、完備性の
一形態です。

## 例題

**問題.** 二分法で $`f(x) = x^2 - 2`$ の零点を $`[1, 2]`$ の中に求め、区間縮小法が何を
保証するかを説明しなさい。

$`I_1 = [1,2]`$ から始めます：$`f(1) = -1 < 0`$, $`f(2) = 2 > 0`$。中点で評価し、$`f`$ が
符号を変える側の半分を残します。
$$
\begin{aligned}
&f(1.5) = 0.25 > 0 &&\Rightarrow I_2 = [1,\ 1.5]\\
&f(1.25) = -0.4375 < 0 &&\Rightarrow I_3 = [1.25,\ 1.5]\\
&f(1.375) = -0.109\ldots < 0 &&\Rightarrow I_4 = [1.375,\ 1.5]\\
&\vdots
\end{aligned}
$$
各段階で長さが半分になります：$`b_n - a_n = \dfrac{1}{2^{\,n-1}} \to 0`$、区間は入れ子
かつ閉です。定理により共通部分はただ1点 $`\xi`$ で、$`a_n \to \xi`$, $`b_n \to \xi`$。
$`f(a_n) \le 0 \le f(b_n)`$ で $`f`$ は連続なので、極限をとると $`f(\xi) = 0`$、すなわち
$`\xi = \sqrt2`$ です。

**答え.** 区間縮小法は $`[1,2]`$ 内にただ1つの $`\xi = \sqrt2`$ を確定させます。二分法は
それを任意の精度で計算する構成的な方法です。

## よくある間違い

- **開区間や半開区間を使う。** 閉であることは本質的です。極限点が欠けた端点から
  逃げてしまいます。
- **一意性のための長さの条件を忘れる。** $`b_n - a_n \to 0`$ がないと共通部分が区間
  全体になり得ます（例：すべての $`n`$ で $`I_n = [0,1]`$ なら $`\bigcap = [0,1]`$）。
- **共通点が有理数だと思い込む。** 捕らえられる $`\xi`$ は一般に無理数です。だからこそ
  $`\mathbb{Q}`$ ではこの定理が成り立ちません。

## 練習問題

1. 終端する小数の端点だけを用いて、共通点が $`\tfrac13`$ となる、$`b_n - a_n \to 0`$ の
   入れ子閉区間を作りなさい。
2. $`I_n`$ が開区間だと証明はどこで破れますか。その手順を正確に指摘しなさい。
3. 区間縮小法（長さの条件つき）が「有界な増加数列は収束する」を導くことを示しなさい。
   *（ヒント：項と上界から区間を作る。）*

<details>
<summary>解答</summary>

1. $`\tfrac13 = 0.3333\ldots`$；$`I_n = [0.\underbrace{3\cdots3}_{n}, \;
   0.\underbrace{3\cdots3}_{n} + 10^{-n}]`$、例えば $`[0.3,0.4], [0.33,0.34], \dots`$。
   長さ $`10^{-n}\to 0`$、入れ子で、各区間に $`\tfrac13`$ が入ります。
2. 端点は依然収束します（$`a_n\uparrow\xi`$, $`b_n\downarrow\eta`$）が、開区間
   $`(a_n,b_n)`$ では共通点 $`\xi=\eta`$ がどの $`(a_n,b_n)`$ にも入るとは限りません。
   包含 $`[\xi,\eta]\subseteq I_n`$ は閉端点を使っていました。よって共通部分が空に
   なり得ます。
3. 上に $`M`$ で有界な増加数列 $`(x_n)`$ に対し、$`[x_1, M]`$ を二分し、数列のしっぽを含む
   側の半分を常に残すと、入れ子区間が得られ、その唯一の点が $`\sup x_n = \lim x_n`$ に
   なります。（$`I_n=[x_n,M]`$ では長さが縮むとは限らないので、この二分の作り方が
   要点です。）

</details>
