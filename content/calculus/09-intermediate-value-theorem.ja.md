# 中間値の定理

## なぜ重要か

中間値の定理（IVT）は、「連続なグラフは、ある高さの下から上へ移るとき必ずそこを
横切らねばならない」という直感を厳密に述べたものです。これは初等解析の主力となる
存在定理です。方程式に解があること（$x^2 = 2$, $\cos x = x$）を証明し、奇数次多項式の
実根を保証し、二分法の根拠となります。決定的に重要なのは、ここで **連続性と完備性が
出会う** ことです。どちらが欠けても定理は成り立ちません。

## 主な結果

> **定理（中間値の定理）.** $f$ を閉区間 $[a, b]$ 上で連続とし、$y$ を $f(a)$ と
> $f(b)$ の間の任意の値とする。このとき $f(c) = y$ となる $c \in [a, b]$ が存在する。

同値に（最もよく使う特別な場合）：$f$ が $[a,b]$ で連続で、$f(a)$ と $f(b)$ が
**異符号** なら、$f$ は $(a, b)$ に根をもつ。

*証明（二分法・区間縮小法）.* $f$ を $f - y$（または $-f$）に置き換えて、
$f(a) < 0 < f(b)$ とし根を探すとしてよい。$a_0 = a$, $b_0 = b$ とし、中点
$m = \tfrac{a_n + b_n}{2}$ で繰り返し二分します。
$$
f(m) \le 0 \text{ なら } [a_{n+1}, b_{n+1}] = [m,\ b_n];
\qquad
f(m) > 0 \text{ なら } [a_{n+1}, b_{n+1}] = [a_n,\ m].
$$
各段階で $f(a_n) \le 0 \le f(b_n)$ であり、区間は入れ子かつ閉で、
$b_n - a_n = \dfrac{b-a}{2^n} \to 0$。[区間縮小法](./05-nested-intervals.ja.md)により
これらはただ1点 $c$ を共有し、$a_n \to c$, $b_n \to c$。連続性より
$f(a_n) \to f(c)$, $f(b_n) \to f(c)$。$f(a_n) \le 0$ から $f(c) \le 0$、
$f(b_n) \ge 0$ から $f(c) \ge 0$。ゆえに $f(c) = 0$。$\blacksquare$

この証明には2本の柱が現れています。**連続性** が符号の情報を極限へ伝え、**完備性**
（区間縮小法を通じて）が極限点 $c$ の存在を保証します。

## 2つの仮定はどちらも本質的

- **連続性は外せない.** $[-1, 1]$ 上で $x < 0$ なら $f(x) = -1$、$x \ge 0$ なら
  $f(x) = 1$ とすると、$-1$ から $1$ へ飛ぶのに決して $0$ になりません。
- **完備性は外せない.** $\mathbb{Q}$ 上で $f(x) = x^2 - 2$ は連続で
  $f(1) < 0 < f(2)$ ですが、**有理数の根をもちません** — 横切る点 $\sqrt2$ がすき間
  だからです。IVT には本当に $\mathbb{R}$ が必要です。

IVT が与えるのは *存在* であり、一意性や公式ではありません。そのような $c$ は複数
あり得て、定理はどれも特定しません。（ただし二分法はその1つを任意の精度で計算します。）

## 例題

**問題.** $\cos x = x$ が $[0, 1]$ に解をもつことを示しなさい。

$f(x) = \cos x - x$ とおくと至る所連続です。端点で、
$$
f(0) = \cos 0 - 0 = 1 > 0, \qquad
f(1) = \cos 1 - 1 \approx 0.540 - 1 = -0.460 < 0.
$$
$f$ は $[0,1]$ で連続で符号を変えるので、IVT により $f(c) = 0$ となる
$c \in (0,1)$、すなわち $\cos c = c$ が存在します。

**答え.** $(0,1)$ に解が存在します。（これは *ドッティ数* $c \approx 0.739$ ですが、
IVT は存在のみを主張します。値を求めるにはさらに作業が必要で、例えば
$x \mapsto \cos x$ の反復や二分法を使います。）

有用な系：**奇数次の多項式は必ず実根をもつ**。$p(x)$ は一方の端で $+\infty$、もう一方で
$-\infty$ となり符号を変えるからです。

## よくある間違い

- **連続性が区間 *全体* で必要なことを忘れる.** $[a,b]$ のどこか1か所の飛びでも結論を
  壊し得ます。
- **存在を一意性と読む.** IVT は *少なくとも1つ* の $c$ を主張します。複数あり得ます。
- **誤った箇所で開区間を使う.** 仮定には閉区間 $[a,b]$ が必要（端点が存在するため）で、
  符号変化の形では根は開区間 $(a,b)$ に位置します。
- **$c$ の公式を期待する.** IVT 自体は非構成的です。数値が必要なら二分法と組み合わせ
  ましょう。

## 練習問題

1. $x^3 - x - 1 = 0$ が実根をもつことを示し、長さ $1$ の区間内に位置づけなさい。
2. 任意の連続関数 $f : [0,1] \to [0,1]$ が *不動点* をもつこと、すなわち $f(c) = c$ と
   なる $c$ が存在することを証明しなさい。*（ヒント：$g(x) = f(x) - x$ に IVT を
   適用する。）*
3. $(0,1]$（注：閉でない）上の連続関数で、IVT の符号変化の結論が破れる例を挙げるか、
   挙げられない理由を説明しなさい。

<details>
<summary>解答</summary>

1. $p(x) = x^3 - x - 1$：$p(1) = -1 < 0$, $p(2) = 5 > 0$ なので根は $(1,2)$ に
   存在する。
2. $g(x) = f(x) - x$ は連続で、$g(0) = f(0) \ge 0$, $g(1) = f(1) - 1 \le 0$。IVT に
   より $g(c) = 0$ となる $c$、すなわち $f(c) = c$ がある。（いずれかの不等式が等号
   なら、その端点が既に不動点。）
3. 結論は *2つの端点の間* の符号変化についてのもの。$(0,1]$ には比べるべき左端点が
   ありませんが、$f$ が符号を変える任意の閉部分区間 $[a,b]\subset(0,1]$ には IVT が
   依然適用できます。区間が閉でないことは端点 $0$ を考察から外すだけで、重要なのは
   閉部分区間上での連続性です。

</details>
