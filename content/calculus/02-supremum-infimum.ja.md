# 上限・下限と実数の完備性

## なぜ重要か

実数の連続性（[デデキントの定理](./01-dedekind-cuts.ja.md)）は、実数直線にすき間が
ないことを述べていました。この抽象的な主張には、非常に使いやすい実用的な形があります。
すなわち、**無限に発散しない空でない実数の集合は、必ず最小上界をもつ** というものです。
これは、極限が存在することを証明するために実際に使う道具です。単調数列が収束すること、
連続関数が最大値をとること、中間値の定理が成り立つこと — これらはすべてこの性質から
導かれます。本レッスンでは、有界性の用語を導入し、その鍵となる定理を述べます。

## 定義

$`S \subseteq \mathbb{R}`$ を実数の集合とします。

> **定義（上界・下界）.**
> 数 $`M`$ が $`S`$ の *上界* であるとは、すべての $`x \in S`$ に対して $`x \le M`$ が
> 成り立つことをいいます。数 $`m`$ が $`S`$ の *下界* であるとは、すべての $`x \in S`$ に
> 対して $`m \le x`$ が成り立つことをいいます。$`S`$ が *上に有界* とは上界をもつこと、
> *下に有界* とは下界をもつこと、*有界* とはその両方であることをいいます。

> **定義（上限・下限）.**
> $`S`$ の *上限*（最小上界）$`\sup S`$ とは、$`S`$ の上界 $`\alpha`$ であって、それより小さい
> 数が上界にならないもののことです。*下限*（最大下界）$`\inf S`$ とは、$`S`$ の下界で
> あって、それより大きい数では追い越せないもののことです。

具体的には、$`\alpha = \sup S`$ は次の2条件で特徴づけられます。

```math
\textbf{(i)}\quad \text{すべての } x \in S \text{ に対して } x \le \alpha;
\qquad
\textbf{(ii)}\quad \text{任意の } \varepsilon > 0 \text{ に対し、ある } x \in S
\text{ で } x > \alpha - \varepsilon.
```

条件 (i) は $`\alpha`$ が上界であること、(ii) は $`\alpha`$ より小さいものは上界に
ならないこと — つまり $`\alpha`$ から $`\varepsilon`$ 以内に $`S`$ の要素が必ずある — を
表します。

$`\sup S`$ は $`S`$ に属するとは限りません。$`S = \{x : 0 \le x < 1\}`$ では
$`\sup S = 1 \notin S`$ です。上限が $`S`$ に属する場合、それは最大値です。

## 主な結果

> **定理（ワイエルシュトラス／実数の完備性）.**
> 上に有界な空でない $`\mathbb{R}`$ の部分集合は、$`\mathbb{R}`$ の中に上限をもつ。同様に、
> 下に有界な空でない部分集合は下限をもつ。

*成り立つ理由（概略）.* これはデデキントの連続性を言い換えたものにすぎません。上に
有界な空でない $`S`$ に対し、$`\mathbb{R}`$ を次のように分けます。

```math
B = \{\, b \in \mathbb{R} : b \text{ は } S \text{ の上界} \,\},
\qquad
A = \mathbb{R} \setminus B.
```

どちらも空ではなく（$`S`$ は上に有界なので $`B \neq \varnothing`$、$`S`$ は空でないので
$`S`$ の要素より下にある数が存在し $`A \neq \varnothing`$）、$`A`$ の任意の要素は $`B`$ の
任意の要素より小さくなります。よって $`(A, B)`$ は $`\mathbb{R}`$ の切断です。デデキントの
定理により、この切断はただ1つの実数 $`\alpha`$ で実現され、$`\alpha`$ が $`S`$ の最小上界に
ほかならないことが確かめられます。下限についての主張は、$`-S = \{-x : x \in S\}`$ に
これを適用すれば従います。$`\blacksquare`$

## 二つの立場：公理か定理か

ここまでで完備性は2度現れました。[デデキント切断のレッスン](./01-dedekind-cuts.ja.md)では
「$`\mathbb{R}`$ の切断に型 III が存在しない」という形で、上の節ではそれと同値な「上限の
存在」という形でです。では、これらは結局 **証明されたもの** なのでしょうか、それとも
**仮定されたもの** なのでしょうか。答えは、$`\mathbb{R}`$ をどう手に入れるかによって変わります。

- **構成的な立場。** 実数を $`\mathbb{Q}`$ の切断として **構成** する立場では、完備性は **定理**
  です。デデキントのレッスンで証明したように、$`\mathbb{R}`$ の切断が指し示す境界の数は
  有理数の下組の和集合として直接組み立てられ、それが型 III の不在を保証しました。本節の
  上限の存在は、そこからさらに従います。この道では、すべては $`\mathbb{Q}`$（さらにさかのぼ
  れば集合論）の上に積み上がる定理であり、循環はありません。
- **公理的な立場。** 一方で、$`\mathbb{R}`$ を最初から「完備順序体」と **定義** してしまう立場も
  あります。このとき完備性（上限の存在）は **公理** であり、証明するものではなく要請する
  ものです。「型 III の切断がない」ことと「上に有界な集合に上限がある」ことは論理的に
  同値なので、どちらを公理に選んでも同じ $`\mathbb{R}`$ が得られます（この公理を満たす順序体
  は同型を除いて一意です）。

どちらの立場でも、完備性を **無から** 証明できるわけではない点は同じです。構成的な立場では
$`\mathbb{Q}`$ と集合論を、公理的な立場では完備性そのものを、出発点として認めています。
数学では必ず何らかの土台を仮定する必要があり、完備性も例外ではありません。大切なのは、
$`\mathbb{Q}`$ ではこの性質が **成り立たない**（$`\{x \in \mathbb{Q} : x^2 < 2\}`$ は上に有界でも
有理数の上限をもたない）という一点であり、これこそが $`\mathbb{R}`$ を特徴づけています。

## 例題

**問題.** $`S = \left\{\, 1 - \dfrac{1}{n} : n = 1, 2, 3, \dots \right\}`$ とする。
$`\sup S`$ と $`\inf S`$ を求め、それぞれ $`S`$ に属するかどうかを述べなさい。

要素は $`0, \tfrac12, \tfrac23, \tfrac34, \dots`$ で、$`1`$ に向かって増加します。

- *下界.* 最小の要素は $`1 - \tfrac11 = 0`$（$`n=1`$）で、各項は $`\ge 0`$ です。よって $`0`$ は
  下界かつ $`0 \in S`$ なので、$`\inf S = 0`$ であり、これは **達成されます**（最小値）。
- *上界.* 各項は $`1 - \tfrac1n < 1`$ なので、$`1`$ は上界です。それより小さいものが上界に
  なるでしょうか。任意の $`\varepsilon > 0`$ に対し、$`\tfrac1n < \varepsilon`$ となる $`n`$ を
  選べば $`1 - \tfrac1n > 1 - \varepsilon`$ です。よって $`1`$ より小さい数は $`S`$ を
  抑えられず、特徴づけ (i)–(ii) により $`\sup S = 1`$ です。これは **達成されない** ので、
  $`S`$ は最大値をもちません。

**答え.** $`\inf S = 0 \in S`$（最小値）、$`\sup S = 1 \notin S`$（最大値なし）。

## よくある間違い

- **上限と最大値を混同する。** 最大値は必ず上限ですが、上限は達成されるとは限りません。
  「$`\sup`$ が存在する」は「$`\max`$ が存在する」よりはるかに弱く、はるかに多くの場合に
  成り立ちます。
- **空でないという条件を落とす。** 空集合は *あらゆる* 実数を上界にもつので、*最小* の
  上界をもちません。定理には本当に $`S \neq \varnothing`$ が必要です。
- **$`\mathbb{Q}`$ 上でも成り立つと思う。** 最小上界は $`\mathbb{Q}`$ では存在しないことが
  あります。完備性は $`\mathbb{R}`$ に固有の性質です。
- **条件 (ii) を忘れる。** $`\alpha`$ が *ある* 上界であることを示すのは半分にすぎません。
  それより小さいものが上界にならないことも示す必要があります。

## 練習問題

1. $`T = \left\{\, \dfrac{(-1)^n}{n} : n \ge 1 \right\}`$ の $`\sup`$ と $`\inf`$ を求め、
   それぞれ達成されるかどうかを述べなさい。
2. 証明せよ：$`\alpha = \sup S`$ かつ $`\alpha \notin S`$ ならば、任意の $`\varepsilon > 0`$ に
   対し、区間 $`(\alpha - \varepsilon, \alpha)`$ は $`S`$ の点を無限個含む。
3. $`A, B`$ を上に有界な空でない集合で $`A \subseteq B`$ とする。$`\sup A \le \sup B`$ を示せ。

<details>
<summary>解答</summary>

1. 各項：$`-1, \tfrac12, -\tfrac13, \tfrac14, \dots`$。最大は $`\tfrac12`$（$`n=2`$）なので
   $`\sup T = \tfrac12 \in T`$（達成）。最小は $`-1`$（$`n=1`$）なので $`\inf T = -1 \in T`$
   （達成）。
2. ある $`\varepsilon`$ について $`(\alpha-\varepsilon,\alpha)`$ に有限個しか点がないとすると、
   それらの最大値（と、残りを抑える $`\alpha`$）から $`\alpha`$ より小さい上界が作れてしまい、
   (ii) に矛盾します。（$`\alpha \notin S`$ なので $`\alpha`$ に等しい点はありません。）
3. $`B`$ の任意の上界は（$`A \subseteq B`$ より）$`A`$ も抑えます。特に $`\sup B`$ は $`A`$ の
   上界なので、$`A`$ の *最小* 上界は $`\sup A \le \sup B`$ を満たします。

</details>
