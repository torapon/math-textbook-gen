# 高階微分

## なぜ重要か

導関数 $`f'`$ はそれ自体ふたたび関数なので、もう一度微分できます — そして何度でも。
**第2次導関数** は、変化率そのものがどう変化しているかを測ります — 物理では加速度、幾何
では *凹凸* です。高階の導関数は関数の形について、より細かい情報を符号化し、それらを1点で
すべて集めることが、次のレッスンの Taylor 近似を生み出します。

## 定義と記法

> **定義.** $`f`$ の *第2次導関数* は $`f'' = (f')'`$ であり、帰納的に *第$`n`$次導関数*
> は $`f^{(n)} = \big(f^{(n-1)}\big)'`$、ただし $`f^{(0)} = f`$ とする。

第$`n`$次導関数のよく使う記法:
```math
f^{(n)}(x), \qquad \frac{d^{n} y}{dx^{n}}, \qquad \frac{d^{n}}{dx^{n}}f(x), \qquad \dot{}\,\ddot{}\ (\text{物理では } n=1,2).
```
連続な導関数を $`n`$ 個もつ関数を $`C^{n}`$ といいます。無限回微分可能な関数（$`e^x`$ や
$`\sin x`$ など）は **滑らか**（smooth）で、$`C^{\infty}`$ と書きます。

## 初等関数のパターン

繰り返し微分すると、しばしばパターンに落ち着きます。

- **多項式** は毎回次数を1つ失い、やがて消えます:
  $`n > m`$ になると $`\dfrac{d^{n}}{dx^{n}}x^{m} = 0`$。一般に $`k \le m`$ で
  $`\dfrac{d^{k}}{dx^{k}}x^{m} = \dfrac{m!}{(m-k)!}x^{m-k}`$。
- **指数関数** は不動点です: $`\dfrac{d^{n}}{dx^{n}}e^{x} = e^{x}`$、また
  $`\dfrac{d^{n}}{dx^{n}}e^{ax} = a^{n}e^{ax}`$。
- **正弦と余弦** は周期 $`4`$ で巡回します:
  ```math
  \frac{d^{n}}{dx^{n}}\sin x = \sin\!\Big(x + \frac{n\pi}{2}\Big),
  ```
  すなわち $`\sin \to \cos \to -\sin \to -\cos \to \sin \to \cdots`$。

## 凹凸：$`f''`$ の意味

第2次導関数の符号は、グラフの *曲がり* を表します。

> 区間で $`f'' > 0`$ なら $`f'`$ が増加し、傾きが急になってグラフは **凸**（上に曲がる）。
> $`f'' < 0`$ ならグラフは **凹**（下に曲がる）。凹凸が切り替わる点が **変曲点** です。

これは最適化も鋭くします — 臨界点 $`c`$（$`f'(c) = 0`$）で、$`f''(c) > 0`$ は極小を、
$`f''(c) < 0`$ は極大を示します — *第2次導関数判定法* です。

## 積に対するライプニッツの公式

積の法則は反復すると二項公式になります — $`(a+b)^n`$ との類似に注目してください。

> **定理（ライプニッツの公式）.** $`f, g`$ が $`n`$ 回微分可能ならば、
> ```math
> (fg)^{(n)} = \sum_{k=0}^{n}\binom{n}{k}\,f^{(k)}\,g^{(n-k)}.
> ```

*証明の概略.* $`n`$ についての帰納法 — 第$`n`$次の公式を1回微分し、パスカルの等式
$`\binom{n}{k} + \binom{n}{k-1} = \binom{n+1}{k}`$ で項をまとめます。二項定理と全く
同じです。$`\blacksquare`$

## 例題

**問題.** $`f(x) = x\,e^{x}`$ の $`f''`$ を2通りで求めなさい。

*直接:* $`f' = e^x + x e^x = (1+x)e^x`$、ついで $`f'' = e^x + (1+x)e^x = (2+x)e^x`$。

*ライプニッツで* $`u = x`$（$`u' = 1`$, $`u'' = 0`$）, $`v = e^x`$ とすると、
```math
f'' = \binom{2}{0}u\,v'' + \binom{2}{1}u'\,v' + \binom{2}{2}u''\,v
= x e^x + 2 e^x + 0 = (x + 2)e^x.
```
どちらも $`f''(x) = (x+2)e^x`$。

## よくある間違い

- **$`f^{(n)}`$ をべきと読む.** $`f^{(3)}`$ は第3次 *導関数* であって、$`f^3 = f\cdot f
  \cdot f`$ ではありません。括弧が両者を区別します。
- **$`(fg)'' = f''g''`$.** 誤り — ライプニッツの公式を使います。中間項 $`2f'g'`$ が最も
  忘れられがちです。
- **滑らかさのクラスの混同.** $`C^1`$（連続な導関数1個）は $`C^2`$ を意味しません。例えば
  $`f(x) = x^2\sin(1/x)`$（$`f(0)=0`$）は至る所微分可能ですが、$`f'`$ は $`0`$ で連続では
  ありません。
- **$`f'' = 0`$ の臨界点を変曲点と思い込む.** $`f(x) = x^4`$ は $`f''(0) = 0`$ ですが、
  そこで狭義の極小です。$`f'' = 0`$ のとき判定法は結論を出しません。

## 練習問題

1. $`f(x) = \cos x`$ の最初の4つの導関数を計算し、巡回を確かめなさい。
2. $`\dfrac{d^{n}}{dx^{n}}\,\dfrac{1}{x}`$ の一般式を求めなさい。
3. ライプニッツの公式で $`(x^2 e^x)^{(n)}`$ を計算しなさい。
4. $`f(x) = x^3 - 3x`$ が凸・凹となる区間を定め、変曲点を求めなさい。

<details>
<summary>解答</summary>

1. $`-\sin x,\ -\cos x,\ \sin x,\ \cos x`$ — 4ステップで出発点に戻る（周期 $`4`$）。
2. $`\dfrac{1}{x} = x^{-1}`$ なので $`\dfrac{d^n}{dx^n}x^{-1} = (-1)(-2)\cdots(-n)\,
   x^{-1-n} = \dfrac{(-1)^n\,n!}{x^{n+1}}`$。
3. $`u = x^2`$（$`u'' = 2`$、それ以上は $`0`$）, $`v = e^x`$ で:
   $`(x^2 e^x)^{(n)} = \big[x^2 + 2nx + n(n-1)\big]e^x`$（寄与するのは $`k = 0, 1, 2`$
   のみ）。
4. $`f'' = 6x`$: $`(-\infty, 0)`$ で凹（$`f'' < 0`$）、$`(0, \infty)`$ で凸（$`f'' > 0`$）、
   変曲点は $`x = 0`$。

</details>
