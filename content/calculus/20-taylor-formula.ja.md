# テイラーの公式

## なぜ重要か

[接線](./12-derivative-definition.ja.md)は関数を *1次* で近似します:
$`f(x) \approx f(a) + f'(a)(x-a)`$。**テイラーの公式** はその体系的な強化版です — $`a`$
における高階の導関数を使って、$`a`$ の近くで $`f`$ に一致する任意次数の最良の多項式を作り、
そして決定的に、*誤差の正確な表現* を与えます。これが、電卓が $`e^x`$, $`\sin x`$, $`\ln x`$
を計算する仕組みであり、べき級数への入口です。この公式は、文字どおり平均値の定理を高階に
押し進めたものです。

## テイラー多項式

> **定義.** $`f`$ が $`a`$ で $`n`$ 回微分可能なとき、$`a`$ における *第$`n`$次テイラー
> 多項式* は
> ```math
> P_{n}(x) = \sum_{k=0}^{n}\frac{f^{(k)}(a)}{k!}\,(x - a)^{k}
> = f(a) + f'(a)(x-a) + \frac{f''(a)}{2!}(x-a)^2 + \cdots + \frac{f^{(n)}(a)}{n!}(x-a)^n.
> ```

これは、$`P_n`$ と $`f`$ が $`a`$ で第$`n`$次導関数まで一致するように作られています:
$`k = 0, 1, \dots, n`$ で $`P_n^{(k)}(a) = f^{(k)}(a)`$。1次の場合 $`P_1(x) = f(a) +
f'(a)(x-a)`$ はまさに接線近似です。

## ラグランジュ剰余つきテイラーの定理

近似はどれくらい良いのでしょうか。剰余 $`R_n(x) = f(x) - P_n(x)`$ は、*次の* テイラー項に
似た、しかし導関数を未知の中間点で評価したきれいな閉じた形をもちます。

> **定理（テイラー、ラグランジュ形）.** $`f`$ が $`a`$ と $`x`$ を含む開区間で $`n+1`$ 回
> 微分可能とする。このとき $`a`$ と $`x`$ の間に点 $`\xi`$ が存在して、
> ```math
> f(x) = P_{n}(x) + \frac{f^{(n+1)}(\xi)}{(n+1)!}\,(x - a)^{n+1}.
> ```

$`n = 0`$ ではこれは $`f(x) = f(a) + f'(\xi)(x-a)`$ — まさに
[平均値の定理](./18-mean-value-theorem.ja.md)です。テイラーの定理はその高階一般化です。

## 証明

道具が1つ必要です — *2つの* 関数に対する平均値の定理です。

> **補題（コーシーの一般化平均値の定理）.** $`F, G`$ が $`[a,x]`$ で連続、$`(a,x)`$ で
> 微分可能で、そこで $`G' \neq 0`$ ならば、ある $`\xi \in (a,x)`$ で
> ```math
> \frac{F(x) - F(a)}{G(x) - G(a)} = \frac{F'(\xi)}{G'(\xi)}.
> ```

*補題の証明.* $`\Phi(t) = \big(F(x) - F(a)\big)G(t) - \big(G(x) - G(a)\big)F(t)`$ に
[ロルの定理](./18-mean-value-theorem.ja.md)を適用します。これは $`\Phi(a) = \Phi(x) =
F(x)G(a) - G(x)F(a)`$ を満たします。ある $`\xi`$ で $`\Phi'(\xi) = 0`$ となり、これを
整理すると主張が得られます。$`\blacksquare`$

*定理の証明.* $`x`$ を固定し、テイラー展開を **動く基点 $`t`$ のまわり** で見ます:
```math
F(t) = f(x) - \sum_{k=0}^{n}\frac{f^{(k)}(t)}{k!}\,(x - t)^{k}, \qquad G(t) = (x - t)^{n+1}.
```
端点では $`F(x) = 0`$, $`F(a) = f(x) - P_n(x) = R_n(x)`$、また $`G(x) = 0`$,
$`G(a) = (x-a)^{n+1}`$。$`F`$ の和は微分すると **望遠鏡的に相殺** します — 各項の2つの
部分が隣と打ち消し合い、残るのは
```math
F'(t) = -\frac{f^{(n+1)}(t)}{n!}\,(x - t)^{n}, \qquad G'(t) = -(n+1)(x - t)^{n}.
```
だけです。$`F, G`$ にコーシーの補題を適用すると、$`a`$ と $`x`$ の間の $`\xi`$ で
```math
\frac{F(a) - F(x)}{G(a) - G(x)} = \frac{F'(\xi)}{G'(\xi)}
= \frac{-\,f^{(n+1)}(\xi)\,(x-\xi)^n/n!}{-(n+1)(x-\xi)^n}
= \frac{f^{(n+1)}(\xi)}{(n+1)!}.
```
左辺は $`\dfrac{R_n(x)}{(x-a)^{n+1}}`$ なので、$`R_n(x) = \dfrac{f^{(n+1)}(\xi)}
{(n+1)!}(x-a)^{n+1}`$ となり、主張どおりです。$`\blacksquare`$

## $`a = 0`$ のまわりのおなじみの展開

$`a = 0`$ でのテイラー多項式（歴史的には *マクローリン* 多項式）は、誰もが出会う級数を
再現します:
```math
e^{x} = 1 + x + \frac{x^2}{2!} + \cdots + \frac{x^n}{n!} + R_n,
\qquad
\sin x = x - \frac{x^3}{3!} + \frac{x^5}{5!} - \cdots,
```
```math
\cos x = 1 - \frac{x^2}{2!} + \frac{x^4}{4!} - \cdots,
\qquad
\ln(1+x) = x - \frac{x^2}{2} + \frac{x^3}{3} - \cdots.
```

## 例題：保証された誤差つきで $`e^{0.1}`$ を近似する

$`f(x) = e^x`$, $`a = 0`$, $`n = 2`$ とすると $`P_2(x) = 1 + x + \tfrac{x^2}{2}`$ で、
```math
e^{0.1} \approx P_2(0.1) = 1 + 0.1 + 0.005 = 1.105.
```
剰余は、ある $`\xi \in (0, 0.1)`$ で $`R_2 = \dfrac{e^{\xi}}{3!}(0.1)^3`$。
$`e^{\xi} < e^{0.1} < 1.2`$ なので、厳密な評価
```math
|R_2| < \frac{1.2}{6}(0.001) = 0.0002
```
が得られます。よって $`e^{0.1} = 1.105 \pm 0.0002`$ — 真の値 $`1.10517\ldots`$ は確かに
その範囲にあります。テイラーの定理の強みはまさにこれです — 単なる推定ではなく、*保証つき*
の推定なのです。

## よくある間違い

- **階乗を忘れる.** $`(x-a)^k`$ の係数は $`f^{(k)}(a)/k!`$ であって $`f^{(k)}(a)`$ では
  ありません。
- **$`\xi`$ を既知として扱う.** MVT の点と同様、$`\xi`$ は存在しますが特定されません。
  区間上で $`|f^{(n+1)}|`$ を最大化して $`R_n`$ を *評価する* ために使うもので、決して値を
  「代入する」ものではありません。
- **項を増やせば至る所で精度が上がると思い込む.** 剰余が実際に小さくなることを示す必要が
  あります。関数によっては、テイラー級数が $`a`$ から離れた所で $`f`$ に収束しません。
- **誤った点のまわりで展開する.** すべての導関数は基点 $`a`$ で評価します。別の点の近くを
  近似するには、そこで展開し直す必要があります。

## 練習問題

1. $`a = 0`$ での $`\sin x`$ の3次テイラー多項式を書きなさい。
2. 1次のテイラー公式で $`|\sin x - \sin y| \le |x-y|`$ を再導出しなさい。
3. $`0`$ での $`\sqrt{1+x}`$ の2次テイラー多項式を使って $`\sqrt{1.2}`$ を近似し、誤差を
   評価しなさい。
4. $`0`$ での $`e^x`$ の第$`n`$次テイラー剰余が $`|R_n(x)| \le \dfrac{e^{|x|}\,|x|^{n+1}}
   {(n+1)!} \to 0`$（$`n \to \infty`$、各固定 $`x`$）を満たすことを示しなさい。

<details>
<summary>解答</summary>

1. $`0`$ での $`f', f'', f'''`$ は $`1, 0, -1`$ なので $`P_3(x) = x - \dfrac{x^3}{6}`$。
2. $`n = 0`$ のテイラーは MVT: $`\sin x - \sin y = \cos(\xi)(x - y)`$、$`|\cos\xi| \le 1`$
   で評価が出ます。
3. $`f(x) = (1+x)^{1/2}`$: $`f(0)=1`$, $`f'(0)=\tfrac12`$, $`f''(0) = -\tfrac14`$ なので
   $`P_2(x) = 1 + \tfrac{x}{2} - \tfrac{x^2}{8}`$、$`P_2(0.2) = 1 + 0.1 - 0.005 =
   1.095`$。剰余は $`f'''(x) = \tfrac38(1+x)^{-5/2}`$ を使い、$`[0,0.2]`$ 上 $`\tfrac38`$
   で抑えられるので $`|R_2| \le \tfrac{3/8}{6}(0.2)^3 = 0.0005`$。（真の値 $`\sqrt{1.2}
   = 1.0954\ldots`$。）
4. $`f^{(n+1)}(x) = e^x`$ で、$`0`$ と $`x`$ の間で $`e^{\xi} \le e^{|x|}`$。ゆえに
   $`|R_n(x)| = \dfrac{e^{\xi}}{(n+1)!}|x|^{n+1} \le \dfrac{e^{|x|}|x|^{n+1}}{(n+1)!}`$、
   階乗がべきを圧倒するので $`\to 0`$。よって $`e^x`$ のテイラー級数は至る所 $`e^x`$ に
   収束します。

</details>
