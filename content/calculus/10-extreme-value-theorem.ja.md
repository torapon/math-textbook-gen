# 最大値・最小値の定理

## なぜ重要か

[中間値の定理](./09-intermediate-value-theorem.ja.md)は、$`[a,b]`$ 上の連続関数が
両端の値の間の値を *1つも* 取りこぼさないことを述べていました。その相棒である
**最大値・最小値の定理**（extreme value theorem, EVT）は、連続関数が無限へ逃げ出す
ことなく、最大値と最小値を実際に *達成する* ことを述べます。最も大きくなる点と最も
小さくなる点が存在する、ということです。これは「最大値を求めよ」という問題の背後にある
存在定理です。閉区間上の連続量を最適化する前に、そもそも求めるべき最大値が存在することを
保証してくれるのが EVT です。中間値の定理と同様、これも **実数の完備性** に依拠し、
ここでは[ボルツァーノ・ワイエルシュトラスの定理](./07-accumulation-points.ja.md)を
通じて効いてきます。

## 主結果

> **定理（最大値・最小値の定理）.** $`f`$ が有界閉区間 $`[a, b]`$ 上で連続ならば、$`f`$ は
> $`[a,b]`$ 上で **有界** であり、最大値と最小値を **達成する**。すなわち、ある点
> $`c, d \in [a, b]`$ が存在して、
> ```math
> f(d) \le f(x) \le f(c) \qquad (\text{すべての } x \in [a, b] \text{ に対し})
> ```
> が成り立つ。

ここには2つの主張が束ねられており、行儀の悪い関数ではどちらも破れ得ます。*有界性*
（値が $`\pm\infty`$ に逃げない）と、*達成*（極値が、近づくだけでなく実際に取られる）です。
$`f(c)`$ が **最大値**、$`f(d)`$ が **最小値** で、合わせて *極値* と呼びます。

## 証明

どちらの主張も同じ仕組みで動きます — 注目したい挙動を追いかける数列をとり、
ボルツァーノ・ワイエルシュトラスの定理で収束部分列を取り出し、連続性で仕上げます。

**ステップ1 — $`f`$ は有界.** 背理法のため、$`f`$ が上に有界 *でない* と仮定します。
すると各 $`n`$ に対し、$`f(x_n) > n`$ となる点 $`x_n \in [a,b]`$ がとれます。数列 $`(x_n)`$ は
$`[a,b]`$ にあるので有界であり、[ボルツァーノ・ワイエルシュトラス](./07-accumulation-points.ja.md)
により収束部分列 $`x_{n_k} \to c`$ をもちます。すべての $`k`$ で $`a \le x_{n_k} \le b`$
なので、極限は $`a \le c \le b`$ を満たします — 区間が **閉** であることはここで使います。
$`c`$ での連続性により $`f(x_{n_k}) \to f(c)`$（有限）。しかし $`f(x_{n_k}) > n_k \to \infty`$
となり矛盾。ゆえに $`f`$ は上に有界です。同じ議論を $`-f`$ に適用すれば下にも有界です。

**ステップ2 — 最大値が達成される.** ここで
```math
M = \sup\{\, f(x) : x \in [a,b] \,\}
```
とおきます。ステップ1と[上限性質](./02-supremum-infimum.ja.md)により、これは存在して有限
です。上限の定義から、各 $`n`$ について $`M - \tfrac1n`$ は上界 *ではない* ので、
```math
M - \tfrac1n < f(x_n) \le M, \qquad \text{したがって } f(x_n) \to M
```
となる $`x_n \in [a,b]`$ がとれます。再び $`(x_n)`$ は $`[a,b]`$ にあるので、ボルツァーノ・
ワイエルシュトラスにより $`x_{n_k} \to c \in [a,b]`$ をとります。連続性より $`f(x_{n_k})
\to f(c)`$。一方 $`f(x_{n_k})`$ は $`M`$ に収束する数列の部分列なので $`f(x_{n_k}) \to M`$
でもあります。極限は一意なので $`f(c) = M`$。上限は $`c`$ で達成され、ゆえに正真正銘の
最大値です。最小値は $`-f`$ の最大値として得られます。$`\blacksquare`$

## 3つの仮定はどれも本質的

どれか1つを外すと結論は破れ得ます。

- **閉.** 半開区間 $`(0, 1]`$ 上で $`f(x) = 1/x`$ は連続ですが上に **非有界** — 有界性が
  破れます。有界でも達成は破れ得ます。開区間 $`(0,1)`$ 上の $`f(x) = x`$ は $`\sup = 1`$,
  $`\inf = 0`$ ですが、どちらも取りません。
- **有界.** 閉だが *無限* の区間 $`[0, \infty)`$ 上で $`f(x) = x`$ は連続かつ非有界です。
  また $`\mathbb{R}`$ 上の $`f(x) = \arctan x`$ は有界（値は $`(-\tfrac\pi2, \tfrac\pi2)`$）
  ですが、どちらの限界も達成しません。
- **連続.** $`[0,1]`$ 上で $`x < 1`$ のとき $`f(x) = x`$、$`f(1) = 0`$ と定めます。すると
  $`\sup f = 1`$ は $`x \to 1^-`$ で近づきますが決して達成されません — $`1`$ における
  1点の不連続が最大値を壊します。

各例は2つの仮定を保ちつつ3つ目だけを破っており、3つの仮定がすべて実際に効いていることが
わかります。（「閉 **かつ** 有界」という性質は *コンパクト性* と呼ばれ、この定理の真の
住処です。[次のレッスン](./11-open-and-closed-sets.ja.md)を参照。）

## 例題

**問題.** $`f`$ が $`[a,b]`$ 上で連続、かつすべての $`x`$ で $`f(x) > 0`$ とする。ある定数
$`\delta > 0`$ が存在して、すべての $`x \in [a,b]`$ で $`f(x) \ge \delta`$ となることを
示しなさい — 各点ごとの正値性ではなく、*一様* な正の下界です。

EVT により $`f`$ はある $`d \in [a,b]`$ で最小値を達成します。すべての $`x`$ で
$`f(x) \ge f(d)`$。$`d \in [a,b]`$ なので仮定から $`f(d) > 0`$。$`\delta = f(d) > 0`$ と
おけば、$`[a,b]`$ 上で至る所 $`f(x) \ge \delta`$ です。

**答え.** そのような $`\delta`$ は存在します。（閉区間であることが本質的です。$`(0,1]`$ 上の
正の関数 $`f(x) = x`$ は値が $`0`$ に縮むので、正の下界を *もちません*。またこの結論から
$`1/f`$ が $`[a,b]`$ 上で有界であることも従います — 連続な非零関数で割るときに常用する
事実です。）

## まとめ：IVT と EVT を合わせる

2つの大定理を組み合わせましょう。$`[a,b]`$ 上の連続関数 $`f`$ は最小値 $`m = f(d)`$ と
最大値 $`M = f(c)`$ を達成し（EVT）、中間値の定理により $`f(d)`$ と $`f(c)`$ の間の
*あらゆる* 値を取ります。ゆえに像はちょうど閉区間
```math
f\big([a,b]\big) = [\,m,\ M\,]
```
になります。**有界閉区間の連続写像による像は、ふたたび有界閉区間である** — どちらの
定理だけでも得られない、すっきりした結論です。

## よくある間違い

- **「有界」と「限界を達成する」の混同.** $`\mathbb{R}`$ 上の $`\arctan`$ は有界ですが
  最大値をもちません。達成を引き出すには、有界性だけでなく有界 *閉* 区間が必要です。
- **区間が閉 *かつ* 有界でなければならないことを忘れる.** どちらか一方の不成立だけでも
  結論は沈み得ます（反例参照）。
- **$`c`$ や $`d`$ の位置がわかると期待する.** IVT と同様、EVT は *存在* 定理であり、点を
  名指ししません。極値の位置を求めるのは、後のレッスンの微分（臨界点）の仕事です。
- **最大値が内部で起こると思い込む.** 端点で起こることもあります — 例えば $`[a,b]`$ 上で
  増加する関数は $`b`$ で最大です。

## 練習問題

1. 達成の部分を仮定せず、$`[a,b]`$ 上の連続関数が有界であることを示しなさい — つまり
   ステップ1を自分の言葉で証明し直しなさい。
2. 開区間 $`(0,1)`$ 上で、有界だが上限も下限も達成しない連続関数を1つ挙げなさい。
3. $`f`$ を $`[a,b]`$ 上の連続関数とする。値域 $`f([a,b])`$ が有界集合であり、
   $`\sup f([a,b])`$ と $`\inf f([a,b])`$ がともにその集合に属することを示しなさい。
4. 真偽を判定せよ：$`f`$ が *閉* だが無限の区間 $`[0,\infty)`$ 上で連続かつ有界ならば、
   $`f`$ は上限を達成しなければならない。理由を述べよ。

<details>
<summary>解答</summary>

1. $`f`$ が上に非有界なら、$`f(x_n) > n`$ となる $`x_n \in [a,b]`$ をとる。ボルツァーノ・
   ワイエルシュトラスにより部分列 $`x_{n_k} \to c \in [a,b]`$（閉性で $`c`$ が範囲内に
   留まる）。連続性より $`f(x_{n_k}) \to f(c)`$（有限）で、$`f(x_{n_k}) > n_k \to \infty`$
   に矛盾。下も同様。
2. $`(0,1)`$ 上の $`f(x) = x`$：$`\sup = 1`$, $`\inf = 0`$ で、端点が除かれているため
   どちらの値も取られない。（または $`f(x) = 2x - 1`$、値域 $`(-1,1)`$。）
3. 値域の有界性はまさにステップ1。上限について：EVT により $`f(c) = M := \sup f([a,b])`$
   となる $`c`$ があるので $`M \in f([a,b])`$。下限も同様に達成される。よって両方とも
   値域に属する。
4. **偽.** $`[0,\infty)`$ 上の $`f(x) = \arctan x`$ は連続かつ上に $`\tfrac\pi2`$ で有界
   だが、$`\sup = \tfrac\pi2`$ は決して達成されない。EVT が要求するのは $`f`$ ではなく
   *区間* の有界性である。

</details>
