# 数列とその極限

## なぜ重要か

解析学に現れるほとんどの対象 — 微分、積分、数 $`e`$、無限級数の和 — は、実は
**数列の極限** です。これらを厳密に扱う前に、数の列 $`a_1, a_2, a_3, \dots`$ がある値
$`L`$ に「近づく」とはどういうことかを、すき間のない形で定義する必要があります。
直感的な言い回し *「項が $`L`$ に近づく」* はあいまいすぎます。どれくらい近づくのか、
いつまで近いのか。$`\varepsilon`$–$`N`$ 論法はこれを厳密にします。

## 定義

> **定義（数列）.** 実数の *数列* とは、関数 $`a : \mathbb{N} \to \mathbb{R}`$ のことで、
> $`(a_n)_{n\ge 1}`$ または単に $`(a_n)`$ と書き、$`n`$ における値を第 $`n`$ *項* $`a_n`$ と
> 呼びます。

> **定義（極限・収束）.** 数列 $`(a_n)`$ が実数 $`L`$ に *収束する* とは、次が成り立つこと
> をいいます。
> $$
> \text{任意の } \varepsilon > 0 \text{ に対し、ある } N \in \mathbb{N}
> \text{ が存在して } \; n \ge N \implies |a_n - L| < \varepsilon.
> $$
> このとき $`\displaystyle \lim_{n\to\infty} a_n = L`$ または $`a_n \to L`$ と書き、$`L`$ を
> *極限* と呼びます。そのような $`L`$ が存在しないとき、$`(a_n)`$ は *発散する* といいます。

これは挑戦と応答のゲームとして読めます。相手が許容誤差 $`\varepsilon > 0`$（どれだけ
近づくべきか）を指定します。あなたは、その先では **すべての** 項が $`L`$ から
$`\varepsilon`$ 以内に入るような境目 $`N`$（数列上の位置）を返さなければなりません。
$`\varepsilon`$ がどんなに小さくても必ず答えられるなら、その数列は $`L`$ に収束します。
条件 $`|a_n - L| < \varepsilon`$ は $`L - \varepsilon < a_n < L + \varepsilon`$ を意味し、
第 $`N`$ 項以降、しっぽ全体が $`L`$ を中心とする幅 $`\varepsilon`$ の帯の中に収まる、という
ことです。

## 2つの基本事実

> **定理（極限の一意性）.** 数列の極限は高々1つである。

*証明.* $`a_n \to L`$ かつ $`a_n \to L'`$ で $`L \neq L'`$ と仮定します。
$`\varepsilon = \tfrac{|L - L'|}{2} > 0`$ とおくと、大きい $`n`$ では
$`|a_n - L| < \varepsilon`$ と $`|a_n - L'| < \varepsilon`$ がともに必要になり、
$`|L - L'| \le |L - a_n| + |a_n - L'| < 2\varepsilon = |L - L'|`$ となって矛盾します。
$`\blacksquare`$

> **定理（収束 $`\Rightarrow`$ 有界）.** 収束する数列はすべて有界である。

*証明.* $`a_n \to L`$ のとき、$`\varepsilon = 1`$ をとると、$`n \ge N`$ で $`|a_n - L| < 1`$ と
なる $`N`$ が存在し、それらの項は $`(L-1, L+1)`$ に入ります。残りの有限個の項
$`a_1, \dots, a_{N-1}`$ も有界です。すべての上下界の最大・最小をとれば $`(a_n)`$ が
有界であることがわかります。$`\blacksquare`$

逆は成り立ちません。$`a_n = (-1)^n`$ は有界ですが収束しないので、有界性だけでは
不十分です。（これを救う追加の条件である単調性が、[次のレッスン](./04-monotone-convergence.ja.md)
の主題です。）

## 例題

**問題.** 定義から直接、$`\displaystyle \lim_{n\to\infty} \frac{n+2}{2n+1} = \frac12`$
を証明しなさい。

候補の極限 $`L = \tfrac12`$ との差を評価します。

```math
\left| \frac{n+2}{2n+1} - \frac12 \right|
= \left| \frac{2(n+2) - (2n+1)}{2(2n+1)} \right|
= \frac{3}{2(2n+1)} < \frac{3}{4n}.
```

$`\varepsilon > 0`$ が与えられたとき、これは $`\frac{3}{4n} < \varepsilon`$、すなわち
$`n > \frac{3}{4\varepsilon}`$ になればすぐに $`< \varepsilon`$ となります。そこで整数 $`N`$ を
$`N > \frac{3}{4\varepsilon}`$ にとります。すると $`n \ge N`$ なるすべての $`n`$ で、

```math
\left| \frac{n+2}{2n+1} - \frac12 \right| < \frac{3}{4n} \le \frac{3}{4N} < \varepsilon.
```

定義により、極限は $`\tfrac12`$ です。$`\blacksquare`$

戦略に注目してください。$`|a_n - L|`$ を **より簡単な** 式（$`\tfrac{3}{4n}`$）で上から
抑え、それを $`n`$ について解きます。最小の有効な $`N`$ を求める必要はなく、うまくいく
$`N`$ ならどれでもかまいません。

## よくある間違い

- **量化子の順序を入れ替える。** 順序は「任意の $`\varepsilon`$ に対し、ある $`N`$」です。
  $`N`$ は $`\varepsilon`$ に依存してよく（$`\varepsilon`$ が小さいほど普通 $`N`$ は大きく
  なります）、すべての $`\varepsilon`$ に共通の $`N`$ を要求するのは別の、はるかに強い
  条件です。
- ***ある* $`n`$ について $`|a_n - L| < \varepsilon`$ を要求する。** これは $`n \ge N`$ なる
  **すべての** $`n`$、つまりしっぽ全体について成り立たねばならず、たまたま1つの項では
  だめです。
- **有界なら収束すると思い込む。** そうではありません。$`(-1)^n`$ を見てください。
- **最適な $`N`$ を探す。** 無駄です。$`|a_n - L|`$ をより簡単なもので過大評価するのが、
  標準的で完全に厳密な近道です。

## 練習問題

1. 定義を用いて、$`\displaystyle \lim_{n\to\infty} \frac{1}{n^2} = 0`$ を証明しなさい。
2. $`a_n = (-1)^n`$ が発散することを証明しなさい。*（ヒント：$`a_n \to L`$ と仮定し、
   $`\varepsilon = 1`$ をとる。）*
3. $`a_n \to L`$ かつ すべての $`n`$ で $`a_n \ge 0`$ ならば $`L \ge 0`$ であることを示しなさい。

<details>
<summary>解答</summary>

1. $`\left|\frac{1}{n^2} - 0\right| = \frac{1}{n^2} \le \frac1n < \varepsilon`$ は
   $`n > \frac1\varepsilon`$ のとき成り立つので、$`N > \frac1\varepsilon`$ ととる。
2. $`a_n \to L`$ なら、$`\varepsilon = 1`$ と大きい $`n`$ で $`|1 - L| < 1`$ と $`|-1 - L| < 1`$ が
   ともに成り立ち、$`|1-(-1)| = 2 \le |1-L| + |{-1}-L| < 2`$ となって不可能。
3. $`L < 0`$ とすると $`\varepsilon = -L > 0`$ をとれて、大きい $`n`$ で
   $`a_n < L + \varepsilon = 0`$ となり $`a_n \ge 0`$ に矛盾。ゆえに $`L \ge 0`$。（なお
   $`a_n > 0`$ と狭めても $`L \ge 0`$ までしか言えません。例：$`a_n = \frac1n`$。）

</details>
