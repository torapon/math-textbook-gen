# 逆関数の微分

## なぜ重要か

多くの関数は、すでに理解している関数の **逆関数** として定義されます。$`\sqrt{x}`$ は
2乗を、$`\ln x`$ は $`e^x`$ を、$`\arcsin`$ や $`\arctan`$ は三角関数を「元に戻し」ます。
各々を一から微分する代わりに、1つのきれいな法則が、*どんな* 逆関数の導関数も元の関数で
表してくれます。幾何的にはあざやかです — $`f^{-1}`$ のグラフは $`f`$ のグラフを直線
$`y = x`$ について鏡映したもので、鏡映は傾きをその **逆数** に変えます。

## 逆関数はいつ存在するか

まず、逆関数がそもそも関数になる条件を思い出します。区間 $`I`$ 上で **連続かつ狭義単調**
な関数は単射であり、[中間値の定理](./09-intermediate-value-theorem.ja.md)によりその像は
区間 $`J`$ になります。よって逆関数 $`f^{-1} : J \to I`$ をもち、これ自身も連続かつ狭義
単調です。（これこそ $`\ln`$ が $`e^x`$ の逆関数として現れた仕組みです。）存在が片づいた
ので、微分します。

## 法則

> **定理（逆関数の微分）.** $`f`$ を区間上で連続かつ狭義単調とし、$`a`$ で微分可能で
> $`f'(a) \neq 0`$ とする。$`g = f^{-1}`$ とおく。このとき $`g`$ は $`b = f(a)`$ で微分
> 可能で、
> ```math
> g'(b) = \frac{1}{f'(a)} = \frac{1}{f'\big(g(b)\big)}.
> ```

ライプニッツの記法では単に $`\dfrac{dx}{dy} = \dfrac{1}{\,dy/dx\,}`$ — 逆関数の導関数は
元の導関数の逆数です。

*証明.* $`k \neq 0`$ をとり、$`y = b + k`$, $`x = g(y)`$ とおくと $`a = g(b)`$ であり、
$`f(x) = y`$, $`f(a) = b`$。$`g`$ は単射なので $`y \neq b`$ から $`x \neq a`$ が従い、
$`g`$ は連続なので $`k \to 0`$ で $`x \to a`$。差分商をひっくり返します。
```math
\frac{g(b+k) - g(b)}{k} = \frac{x - a}{f(x) - f(a)} = \left(\frac{f(x) - f(a)}{x - a}\right)^{-1}.
```
$`k \to 0`$ のとき $`x \to a`$ なので、内側の商は $`f'(a) \neq 0`$ に近づき、その逆数は
$`1/f'(a)`$ に近づきます。$`\blacksquare`$

$`g`$ が微分可能だとわかってしまえば、1行の *記憶法* もあります — 恒等式 $`f(g(y)) = y`$
に[連鎖律](./16-chain-rule.ja.md)を適用すると $`f'(g(y))\,g'(y) = 1`$、ゆえに
$`g'(y) = 1/f'(g(y))`$。上の証明は、そもそも微分してよいことを *正当化* するものです。

**破綻する場所.** $`f'(a) = 0`$ なら $`f`$ の接線は水平なので、鏡映された $`g`$ の接線は
*垂直* になり、$`g`$ は $`b`$ で微分不可能です。例：$`f(x) = x^3`$ は $`f'(0) = 0`$ で、
その逆関数 $`g(y) = y^{1/3}`$ は $`0`$ で垂直接線をもちます。

## 例

**平方根.** $`g(x) = \sqrt{x}`$ は $`[0,\infty)`$ 上で $`f(x) = x^2`$ を逆にし、
$`f'(x) = 2x`$。よって $`x > 0`$ で、
```math
g'(x) = \frac{1}{f'(g(x))} = \frac{1}{2\sqrt{x}},
```
べきの法則 $`\tfrac{d}{dx}x^{1/2} = \tfrac12 x^{-1/2}`$ と一致します。

**対数、再訪.** $`\ln`$ は $`f(x) = e^x`$（$`f' = e^x`$）を逆にします。ゆえに
```math
(\ln)'(y) = \frac{1}{e^{\ln y}} = \frac1y,
```
以前と同じ答えが、今度は逆関数の法則だけから得られます。

**逆正弦.** $`y = \arcsin x`$ は $`x = \sin y`$（$`y \in [-\tfrac\pi2, \tfrac\pi2]`$、
そこで $`\cos y \ge 0`$）を意味します。$`\sin' = \cos`$ なので、
```math
\frac{d}{dx}\arcsin x = \frac{1}{\cos y} = \frac{1}{\sqrt{1 - \sin^2 y}} = \frac{1}{\sqrt{1 - x^2}}
\qquad (-1 < x < 1).
```

**逆正接.** $`y = \arctan x`$ は $`x = \tan y`$ を意味します。$`\tan' = \sec^2 = 1 +
\tan^2`$ なので、
```math
\frac{d}{dx}\arctan x = \frac{1}{\sec^2 y} = \frac{1}{1 + \tan^2 y} = \frac{1}{1 + x^2}.
```

## よくある間違い

- **$`f'(a) = 0`$ の所で法則を使う.** 水平な接線は垂直な接線に鏡映され、そこで逆関数は
  微分不可能です。
- **$`f'`$ を誤った点で評価する.** 逆数は $`1/f'(g(b))`$ です — 逆関数の *出力* を
  $`f'`$ に戻して代入します。$`b`$ そのものではありません。
- **$`(f^{-1})'`$ と $`(1/f)'`$ の混同.** 逆関数の導関数は、逆数 $`1/f`$ の導関数では
  ありません。両者は無関係です。
- **定義域の制限を落とす.** $`\arcsin'`$ は $`(-1,1)`$ で成り立ちます。$`\arcsin`$ 自身は
  $`[-1,1]`$ で定義されますが、端点では微分不可能です。

## 練習問題

1. $`f(x) = x^n`$ を用いて、$`x > 0`$ で $`\dfrac{d}{dx}x^{1/n} = \dfrac1n x^{1/n - 1}`$
   を導きなさい。
2. $`\dfrac{d}{dx}\arccos x`$ を求め、$`\dfrac{d}{dx}\arcsin x`$ と比べなさい。
3. $`f(x) = x + e^x`$ とする。$`f`$ が微分可能な逆関数 $`g`$ をもつことを示し、$`g'(1)`$
   を計算しなさい。（ヒント：$`f(0) = 1`$。）
4. $`f(x) = x^3 - 3x`$ の逆関数はどこで微分不可能ですか。（ヒント：$`f'(x) = 0`$ を
   解く。）

<details>
<summary>解答</summary>

1. $`f'(x) = n x^{n-1}`$, $`g(x) = x^{1/n}`$ なので $`g'(x) = \dfrac{1}{n\,(x^{1/n})^{n-1}}
   = \dfrac1n x^{-(n-1)/n} = \dfrac1n x^{1/n - 1}`$。
2. $`y = \arccos x`$, $`x = \cos y`$, $`\cos' = -\sin`$、$`[0,\pi]`$ 上で $`\sin y \ge 0`$
   なので $`\dfrac{d}{dx}\arccos x = \dfrac{1}{-\sin y} = -\dfrac{1}{\sqrt{1-x^2}}`$ —
   $`\arcsin'`$ の符号反転。（実際 $`\arcsin x + \arccos x = \tfrac\pi2`$。）
3. $`f'(x) = 1 + e^x > 0`$ なので $`f`$ は狭義単調増加かつ連続で、$`f' \neq 0`$ の所
   — すなわち至る所 — で微分可能な逆関数をもちます。$`f(0) = 1`$ なので $`g(1) = 0`$、
   $`g'(1) = 1/f'(0) = 1/(1 + 1) = \tfrac12`$。
4. $`f'(x) = 3x^2 - 3 = 0`$ は $`x = \pm 1`$ で、点 $`f(\pm 1) = \mp 2`$ を与えます。逆関数
   は $`y = \mp 2`$ で垂直接線をもち（微分不可能）。（なお $`f`$ はここでは大域的には
   可逆でないので、単調な部分に制限します。）

</details>
