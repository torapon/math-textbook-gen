# 導関数：定義と微分可能性

## なぜ重要か

連続性は「関数に飛びがない」ことを教えてくれました。**導関数** はもっと鋭い問いを
立てます — 関数は *どれだけ速く* 変化しているのか、そしてグラフは *どちらを向いて* いる
のか。その答えが接線の傾き、すなわち、ある点における曲線への最良の直線近似です。区間に
わたって平均すれば「傾き」は単なる（縦の変化）÷（横の変化）ですが、導関数は、その平均を
区間が1点へ縮むときの極限としてとらえたものです。これ以降のすべての考え — 最適化、平均値
の定理、Taylor 近似 — は、この1つの極限の上に築かれます。

## 差分商と導関数

定義域の点 $`a`$ を固定します。$`a`$ から近くの点までの区間で、$`f`$ の **平均変化率** は
割線の傾きです。
```math
\frac{f(a + h) - f(a)}{h} \qquad (h \neq 0).
```
これを *差分商* といいます。2つ目の点を $`a`$ に滑り込ませる — つまり $`h \to 0`$ と
する — と、極限が存在するなら割線は接線になります。

> **定義（導関数）.** $`f`$ が $`a`$ で *微分可能* であるとは、極限
> ```math
> f'(a) = \lim_{h \to 0} \frac{f(a + h) - f(a)}{h}
> ```
> が（有限の値として）存在することをいう。$`x = a + h`$ と置き換えれば、同値に
> ```math
> f'(a) = \lim_{x \to a} \frac{f(x) - f(a)}{x - a}.
> ```
> 値 $`f'(a)`$ を $`a`$ における $`f`$ の *微分係数*（導関数の値）という。$`f`$ が開区間の
> すべての点で微分可能なら、対応 $`x \mapsto f'(x)`$ を *導関数* $`f'`$ という。

差分商はつねに $`h \neq 0`$（あるいは $`x \neq a`$）でのみ評価される点に注意してください。
$`0`$ で割ることは決してありません。極限はいつものとおり、商が禁じられた点でとる値ではなく、
商が *近づく* 先についての話です — まさに[関数の極限の定義](./08-functions-and-continuity.ja.md)
が用意された状況です。

> **記法.** 同じ導関数はいくつかの書き方をします。
> ```math
> f'(a), \qquad \frac{df}{dx}\bigg|_{x=a}, \qquad \frac{dy}{dx}, \qquad \dot{y}.
> ```
> ラグランジュの $`f'`$、ライプニッツの $`\tfrac{dy}{dx}`$（実際の商ではなく1つの記号
> として読みます — とはいえ商のように振る舞います）、ニュートンの $`\dot y`$（時間微分で
> よく使う）は、すべて同じものを表します。

## 接線と1次近似

幾何的には、$`f'(a)`$ は点 $`\big(a, f(a)\big)`$ におけるグラフの **接線** の傾きです。
```math
y = f(a) + f'(a)\,(x - a).
```
これを近似として読むと、微分可能性の最も有用な帰結が得られます — $`a`$ の近くでは、曲線は
ほとんど接線そのものです。
```math
f(x) \approx f(a) + f'(a)\,(x - a) \qquad (x \text{ は } a \text{ の近く}).
```
「微分可能」とは、まさにこのような *1次* の近似が存在し、1次の精度で正確であることを
意味します。これが Taylor の公式の種です。

## 例題

**例1 — べき.** $`f(x) = x^2`$ とすると、
```math
\frac{(a+h)^2 - a^2}{h} = \frac{2ah + h^2}{h} = 2a + h \xrightarrow{\ h \to 0\ } 2a,
```
ゆえに $`f'(a) = 2a`$。（極限の内側では $`h \neq 0`$ なので $`h`$ で割れました。）

**例2 — 逆数.** $`x \neq 0`$ で $`f(x) = \dfrac1x`$ とすると、
```math
\frac{1}{h}\left(\frac{1}{a+h} - \frac1a\right)
= \frac{1}{h}\cdot\frac{a - (a+h)}{a(a+h)}
= \frac{-1}{a(a+h)} \xrightarrow{\ h \to 0\ } -\frac{1}{a^2}.
```
よって $`\dfrac{d}{dx}\,\dfrac1x = -\dfrac{1}{x^2}`$。

## 微分可能ならば連続

微分可能性は連続性より *強い* 条件です — 関数が飛ぶところでは成り立ちません。

> **定理.** $`f`$ が $`a`$ で微分可能ならば、$`f`$ は $`a`$ で連続である。

*証明.* $`x \neq a`$ に対して、次の恒等式を書きます。
```math
f(x) - f(a) = \frac{f(x) - f(a)}{x - a}\,\cdot\,(x - a).
```
$`x \to a`$ のとき、第1因子は $`f'(a)`$（$`f`$ は微分可能なので有限）に、第2因子は $`0`$ に
近づきます。[極限の演算](./08-functions-and-continuity.ja.md)により、積は
$`f'(a) \cdot 0 = 0`$ に近づきます。ゆえに $`f(x) - f(a) \to 0`$、すなわち
$`f(x) \to f(a)`$ となり、これは $`a`$ での連続性です。$`\blacksquare`$

## 逆は成り立たない

連続性は微分可能性を **含意しません**。標準的な例は $`a = 0`$ における絶対値
$`f(x) = |x|`$ です。差分商は
```math
\frac{|0 + h| - |0|}{h} = \frac{|h|}{h} = \mathrm{sign}(h)
```
であり、$`h > 0`$ で $`+1`$、$`h < 0`$ で $`-1`$ です。右側極限と左側極限が食い違うので極限は
存在しません。$`|x|`$ は至る所で連続ですが $`0`$ では微分不可能です。幾何的にはグラフが
**角（かど）** をもち、接線が1本に定まりません。（さらに劇的には、ワイエルシュトラスは
*すべての* 点で連続なのに *どの* 点でも微分不可能な関数を作りました。連続性は本当に
より弱い条件なのです。）

## よくある間違い

- **連続 $`\Rightarrow`$ 微分可能 と思い込む.** $`|x|`$ のような角は連続ですがそこで導関数を
  もちません。含意は一方向だけです。
- **両側の片側極限が一致する必要を忘れる.** 導関数が存在するのは、差分商が両側から
  *同じ* 極限をもつときだけです。
- **$`f'(a)`$ と $`f'`$ の混同.** $`f'(a)`$ は数（1点での傾き）、$`f'`$ はそれらの傾きを
  まとめた関数です。
- **$`0`$ で割る心配をする.** 商は $`h \neq 0`$ でとり、その後に極限が $`h \to 0`$ を
  扱います。極限に移る前に自由に約分してかまいません。
- **$`\tfrac{dy}{dx}`$ を分数と読む.** これは極限を表す1つの記号です。（連鎖律のおかげで
  しばしば分数のように *操作* されますが、分数ではありません。）

## 練習問題

1. 定義から、$`f(x) = x^3`$ が $`f'(a) = 3a^2`$ をもつことを示しなさい。
2. 定義から、$`a > 0`$ で $`f(x) = \sqrt{x}`$ が $`f'(a) = \dfrac{1}{2\sqrt a}`$ を
   もつことを示しなさい。
3. $`f(x) = x\,|x|`$ が $`0`$ で微分可能であることを示し、$`f'(0)`$ を求めなさい（式が
   $`0`$ で符号を変えるにもかかわらず）。
4. $`f(x) = |x^2 - 1|`$ はどの点で微分不可能ですか。理由を述べなさい。

<details>
<summary>解答</summary>

1. $`\dfrac{(a+h)^3 - a^3}{h} = \dfrac{3a^2 h + 3a h^2 + h^3}{h} = 3a^2 + 3ah + h^2
   \to 3a^2`$。
2. $`\dfrac{\sqrt{a+h} - \sqrt a}{h} = \dfrac{(a+h) - a}{h\big(\sqrt{a+h} + \sqrt a\big)}
   = \dfrac{1}{\sqrt{a+h} + \sqrt a} \to \dfrac{1}{2\sqrt a}`$（分子を有理化）。
3. $`\dfrac{h\,|h| - 0}{h} = |h| \to 0`$ なので $`f'(0) = 0`$。（実際 $`f(x) = x|x|`$ は
   至る所で微分可能で $`f'(x) = 2|x|`$。$`0`$ では2つの片側傾きがともに $`0`$ なので角は
   ありません。）
4. $`x = \pm 1`$ で。そこで $`x^2 - 1`$ が $`0`$ を横切るため、$`|x^2 - 1|`$ は折り返して
   角を作ります（片側傾き $`\mp 2`$ と $`\pm 2`$ が食い違う）。それ以外では $`f`$ は
   $`\pm(x^2 - 1)`$ の滑らかな部分であり微分可能です。

</details>
