# デデキント切断と実数の連続性

## なぜ重要か

有理数 $`\mathbb{Q}`$ は一見びっしり詰まっています。どの2つの有理数の間にも、必ず別の
有理数があるからです。ところが、有理数には目に見えない **すき間（穴）** がたくさん
あります。$`2`$ 乗して $`2`$ になる有理数は存在しませんが、$`2`$ 乗が $`2`$ にいくらでも近い
有理数なら見つけられます。$`\mathbb{Q}`$ の上で微積分をしようとすると、本来「存在する
はず」の極限が、こうしたすき間からこぼれ落ちてしまうのです。

実数 $`\mathbb{R}`$ は、まさに *すき間をなくす* ために作られます。この考えを厳密にする
ことが本レッスンの目標です。デデキントの着想は、数を小数展開ではなく、
**有理数全体を左側と右側にどう分けるか** によって特徴づける、という点にあります。

## 定義

> **定義（切断）.** $`\mathbb{Q}`$ の *切断* とは、有理数の集合の組 $`(A, B)`$ であって、
> 次を満たすもののことです。
>
> 1. $`A`$ と $`B`$ はともに空でなく、合わせて $`\mathbb{Q}`$ 全体になる：
>    $`\; A \cup B = \mathbb{Q}`$, $`\; A \cap B = \varnothing`$;
> 2. $`A`$ の任意の要素は $`B`$ の任意の要素より小さい：$`a \in A`$ かつ $`b \in B`$ ならば
>    $`a < b`$。

$`A`$ を *下組*、$`B`$ を *上組* と呼びます。直感的には、切断は数直線上のある1点を指し示し、
その左側がすべて $`A`$、右側がすべて $`B`$ に入る、というイメージです。

## 切断の3つの型

はじめに、これから用いる2つの言葉を説明します。集合の *最大元* とは、その集合に
属する要素であって、**かつ** その集合のすべての要素以上であるもののことです。*最小元*
も同様に、その集合に属し、すべての要素以下である要素のことです。有限集合なら必ず
どちらも存在しますが、無限集合では存在するとは **限りません**。たとえば
$`\{x \in \mathbb{Q} : x < 1\}`$ には最大元がありません。どの要素をとっても、それと
$`1`$ の間にさらに別の要素があるからです。この「存在するとは限らない」という点こそが、
次に述べる3つの型を分ける鍵になります。

$`\mathbb{Q}`$ は **稠密な順序** をもつ（どの2つの有理数の間にも第3の有理数がある）ので、
論理的に考えられる場合のうち1つは直ちに排除されます。$`A`$ が最大元 $`a^{*}`$ をもち、
*同時に* $`B`$ が最小元 $`b^{*}`$ をもつことはあり得ません。もしそうなら有理数
$`\frac{a^{*} + b^{*}}{2}`$ がどちらの組にも属さなくなるからです。残るのはちょうど3つの型です。

| 型 | 下組 $`A`$ | 上組 $`B`$ | 意味 |
|:----:|:----------------|:----------------|:--------|
| I  | 最大元をもつ | 最小元を **もたない** | 切断は *ある有理数の所* |
| II | 最大元を **もたない** | 最小元をもつ | 切断は *ある有理数の所* |
| III | 最大元を **もたない** | 最小元を **もたない** | 切断は **すき間** にある |

型 I と型 II は、いずれも境界となる有理数1つに切断を対応させます。面白いのは型 III で、
これは **有理数が存在しない** 場所を指しています。このようなすき間こそが、これから
*無理数* と呼ぶものです。

## 例題：$`\sqrt{2}`$ を定める切断

**問題.** 次の組

```math
A = \{\, x \in \mathbb{Q} : x \le 0 \ \text{または}\ x^2 < 2 \,\}, \qquad
B = \{\, x \in \mathbb{Q} : x > 0 \ \text{かつ}\ x^2 > 2 \,\}
```

が $`\mathbb{Q}`$ の **型 III** の切断であることを示しなさい。

まず $`(A,B)`$ が確かに切断であることを確認します。正の有理数はどれも $`x^2 < 2`$,
$`x^2 = 2`$, $`x^2 > 2`$ のいずれかですが、$`x^2 = 2`$ となる有理数は存在しないので、
すべての有理数は $`A, B`$ のちょうど一方に入り、$`a \in A`$ は $`b \in B`$ より小さくなります。

次に $`A`$ が **最大元をもたない** ことを示します。$`a \in A`$ で $`a > 0`$（よって $`a^2 < 2`$）
を任意にとり、$`A`$ に属するもう少し大きい有理数を探します。

```math
a' = a + \frac{2 - a^2}{a + 2}\,= \frac{2a+2}{a+2}
```

とおくと $`a' > a`$ であり、簡単な計算で

```math
a'^2 - 2 = \frac{(2a+2)^2 - 2(a+2)^2}{(a+2)^2}
        = \frac{2(a^2 - 2)}{(a+2)^2} < 0
```

となるので $`a'^2 < 2`$、すなわち $`a' \in A`$ です。したがって $`A`$ のどの要素にも、
より大きい要素が存在し、$`A`$ は最大元をもちません。同様の議論で $`B`$ は **最小元を
もたない** ことが示せます。ゆえにこの切断は型 III であり、私たちが $`\sqrt{2}`$ と
名づけるすき間を指し示しています。$`\blacksquare`$

## 実数の連続性

実数は、まさにこのすき間の問題が起こらないように *定義* されます。その決定的な性質が
次です。

> **デデキントの定理（実数の連続性・完備性）.**
> **実数** の任意の切断 $`(A, B)`$ に対して、それを生み出す実数 $`\alpha`$ がただ1つ
> 存在する。すなわち、$`\alpha`$ は $`A`$ の最大元であるか、または $`B`$ の最小元である。
> **$`\mathbb{R}`$ の切断には型 III が存在しない。**

言い換えれば、実数直線にはすき間がありません。$`\mathbb{R}`$ を左右に分けるどんな
分け方も、切断のちょうどその場所にある実在の数によって実現されます。この一文、
すなわち「型 III の切断が存在しない」ということこそが、**実数の連続性** の意味であり、
解析学のあらゆる極限定理が最終的に拠って立つ土台です。

## デデキントの定理の証明

この定理は、実数を **有理数の切断として構成する** 立場に立てば、定義から証明できます。
鍵となる発想は、すき間に対応する数を **有理数から直接組み立てる** ことです。

まず実数の表し方を1つ固定します。各実数 $`\alpha`$ に対し、それより小さい有理数全体の集合

```math
\ell(\alpha) = \{\, q \in \mathbb{Q} : q < \alpha \,\}
```

を考えます。これは $`\alpha`$ の下側にある有理数を集めた「下組」で、デデキントの構成では
実数 $`\alpha`$ そのものをこの集合と同一視します。$`\ell(\alpha)`$ は空でなく、$`\mathbb{Q}`$ 全体
でもなく、下に閉じており（$`q \in \ell(\alpha)`$ かつ $`q' < q`$ なら $`q' \in \ell(\alpha)`$）、
最大元をもちません。また有理数 $`q`$ について $`q \in \ell(\alpha) \iff q < \alpha`$ であり、
$`\alpha \le \beta \iff \ell(\alpha) \subseteq \ell(\beta)`$ が成り立ちます。

さて $`\mathbb{R}`$ の切断 $`(A, B)`$ を任意にとります（ここで $`A, B`$ は **実数** の集合です）。
次の有理数の集合を作ります。

```math
U = \bigcup_{\alpha \in A} \ell(\alpha)
```

これは $`A`$ に属する実数たちの下組を、すべて重ね合わせたものです。

**$`U`$ がふたたび1つの実数を定めること。** $`U`$ が上の4つの性質をもつ下組であることを
確かめます。$`A`$ は空でなく各 $`\ell(\alpha)`$ も空でないので $`U \neq \varnothing`$。$`B`$ から1つ
$`\beta`$ をとると、任意の $`\alpha \in A`$ について $`\alpha < \beta`$、よって $`\ell(\alpha) \subseteq \ell(\beta)`$
となり、$`U \subseteq \ell(\beta) \neq \mathbb{Q}`$ なので $`U \neq \mathbb{Q}`$。下に閉じていることと
最大元をもたないことは、各 $`\ell(\alpha)`$ がそうであることから和集合へそのまま移ります。
ゆえに $`U = \ell(\gamma)`$ となる実数 $`\gamma`$ がただ1つ定まります。**この $`\gamma`$ こそ、切断が
指し示す境界の数であり、完備性を仮定せず有理数の和集合として手の中に作り出された** ことに
注意してください。

**$`\gamma`$ が境界であること。** 各 $`\alpha \in A`$ について $`\ell(\alpha) \subseteq U = \ell(\gamma)`$ なので
$`\alpha \le \gamma`$。また各 $`\beta \in B`$ について $`U \subseteq \ell(\beta)`$ すなわち
$`\ell(\gamma) \subseteq \ell(\beta)`$ なので $`\gamma \le \beta`$。

**結論。** $`A \cup B = \mathbb{R}`$ なので、実在する $`\gamma`$ も $`A`$ か $`B`$ のちょうど一方に入ります。

- $`\gamma \in A`$ のとき：すべての $`\alpha \in A`$ で $`\alpha \le \gamma`$ かつ $`\gamma \in A`$ なので、
  $`\gamma = \max A`$（型 I）。
- $`\gamma \in B`$ のとき：すべての $`\beta \in B`$ で $`\gamma \le \beta`$ かつ $`\gamma \in B`$ なので、
  $`\gamma = \min B`$（型 II）。

どちらの場合も境界元（最大元または最小元）が存在し、型 III は起こり得ません。$`\blacksquare`$

$`\mathbb{Q}`$ の切断（例題の $`\sqrt{2}`$）で型 III が生じたのは、境界 $`\gamma`$ が $`\mathbb{Q}`$ の外に
落ちて、どちらの組も捕まえられなかったからでした。$`\mathbb{R}`$ では $`\gamma`$ を有理数の
集まりとして必ず構成できるため、同じことは起きません。

## よくある間違い

- **$`A`$ が最大元をもつことを要求してしまう。** 切断は $`A`$ に境界元をもつとは限りません。
  もつかどうかが、まさに型を分ける基準です。
- **型 III の切断を「$`\mathbb{R}`$ で数が欠けている」と考えてしまう。** $`\mathbb{Q}`$ では
  欠けていますが、$`\mathbb{R}`$ では定義によってそのすき間が埋められています。型 III の
  切断は $`\mathbb{Q}`$ には存在しても、$`\mathbb{R}`$ には決して存在しません。
- **稠密性を忘れる。** 「$`A`$ が最大元をもち $`B`$ が最小元をもつ」場合が不可能なのは、
  まさに順序の稠密性によります。稠密性を外すと分類は変わります。

## 練習問題

1. 有理数 $`3`$ に対応する $`\mathbb{Q}`$ の切断を述べなさい。それは何型ですか。また、
   $`3`$ の *もう一方* の表し方（$`A`$ の最大元か $`B`$ の最小元か）も正しい切断ですか。
2. すき間が数 $`1`$ であるような型 III の切断が存在しない理由を説明しなさい。
3. 上の $`\sqrt{2}`$ の切断 $`(A,B)`$ について、任意の $`\varepsilon > 0`$ に対し、
   $`b - a < \varepsilon`$ となる $`a \in A`$, $`b \in B`$ が存在することを示しなさい。

<details>
<summary>解答</summary>

1. $`A = \{x : x < 3\},\, B = \{x : x \ge 3\}`$（型 II、$`\min B = 3`$）か、または
   $`A = \{x : x \le 3\},\, B = \{x : x > 3\}`$（型 I、$`\max A = 3`$）。どちらも正しい
   切断です。1つの有理数は型 I と型 II の2通りの表し方を生みますが、それらを同じ数
   $`3`$ と同一視します。
2. $`1`$ は有理数なので $`\max A`$（または $`\min B`$）になり、切断は型 I／II であって型 III
   ではありません。すき間は有理数が存在しない場所にだけ生じます。
3. 例題より、正の $`a \in A,\ b \in B`$ に対し $`a^2 < 2 < b^2`$ なので、
   $`b - a < \frac{b^2 - a^2}{a+b} = \frac{(b^2-2)+(2-a^2)}{a+b}`$。$`a^2, b^2`$ が $`2`$ から
   $`\delta`$ 以内になるよう $`a, b`$ を選べば、$`b-a`$ をいくらでも小さくできます。

</details>
