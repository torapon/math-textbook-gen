# 関数・極限・連続性

## なぜ重要か

ここまでは数と数列を学んできました。しかし微積分の主役は **関数** であり、その中心的な
定性的性質が *連続性* です。突然の飛びがない関数、いわば「ペンを離さずに描ける」関数の
ことです。とはいえこの標語は、何かを証明するにはあいまいすぎます。本レッスンでは、
関数の極限と連続性の厳密な $`\varepsilon`$–$`\delta`$ による定義を与え、すでに理解している
数列の極限と結びつけます。

## 定義

> **定義（関数）.** *関数* $`f : D \to \mathbb{R}`$ は、*定義域* $`D \subseteq \mathbb{R}`$ の
> 各 $`x`$ に、ただ1つの実数値 $`f(x)`$ を対応させます。

> **定義（関数の極限）.** $`a`$ を $`D`$ の[集積点](./07-accumulation-points.ja.md)とする。
> $`\displaystyle \lim_{x \to a} f(x) = L`$ であるとは、次が成り立つことをいう。
> ```math
> \text{任意の } \varepsilon > 0 \text{ に対し、ある } \delta > 0 \text{ が存在して }
> 0 < |x - a| < \delta,\ x \in D \implies |f(x) - L| < \varepsilon.
> ```

$`a`$ を $`D`$ の集積点とするのは、$`a`$ にいくらでも近い $`D`$ の点を通って極限をとれる
ようにするためです。$`x = a`$ 自身は除外され（$`0 < |x-a|`$）、値 $`f(a)`$ は — たとえ存在
しても — 極限には無関係です。

> **定義（点における連続性）.** $`f`$ が $`a \in D`$ で *連続* であるとは、
> ```math
> \text{任意の } \varepsilon > 0 \text{ に対し、ある } \delta > 0 \text{ が存在して }
> |x - a| < \delta,\ x \in D \implies |f(x) - f(a)| < \varepsilon
> ```
> が成り立つことをいう。$`f`$ が（$`D`$ 上で）*連続* とは、$`D`$ の各点で連続なことをいう。

$`a`$ が $`D`$ の集積点のとき、$`a`$ での連続性はちょうど
$`\displaystyle \lim_{x\to a} f(x) = f(a)`$ を意味します。極限が存在し、**かつ** 関数値に
等しい、ということです。3つのことが一致しなければなりません — $`f(a)`$ が定義され、
極限が存在し、両者が等しい。

## 数列による特徴づけ

関数の極限と数列の極限は、同じ考えの2つの見方です。

> **定理（数列による判定）.** $`\displaystyle\lim_{x\to a} f(x) = L`$ であることは、
> $`x_n \to a`$ となる $`D \setminus \{a\}`$ 内の **すべての** 数列 $`(x_n)`$ に対して
> $`f(x_n) \to L`$ となることと同値である。したがって、$`f`$ が $`a`$ で連続であることは、
> そのようなすべての数列で $`x_n \to a \Rightarrow f(x_n) \to f(a)`$ となることと同値。

この橋渡しにより、数列について証明したことをすべて持ち込めます。特に、極限は
四則演算と両立します。

> **定理（極限の演算）.** $`\lim_{x\to a} f = L`$, $`\lim_{x\to a} g = M`$ ならば、
> $`x \to a`$ のとき、
> ```math
> f \pm g \to L \pm M, \qquad fg \to LM, \qquad \frac{f}{g} \to \frac{L}{M}\ (M \neq 0).
> ```
> ゆえに連続関数の和・積・商・合成は（定義される範囲で）連続である。多項式は至る所で
> 連続、有理関数は分母が $`0`$ でないところで連続である。

数列による判定は、極限が *存在しない* ことを示す最も簡単な方法でもあります。$`f`$ が
異なる値に近づく2つの数列 $`x_n \to a`$, $`x_n' \to a`$ を見つければよいのです。

## 例題

**問題.** $`f(x) = x^2`$ がすべての $`a \in \mathbb{R}`$ で連続であることを、
$`\varepsilon`$–$`\delta`$ の定義から直接示しなさい。

$`|x^2 - a^2| = |x - a|\,|x + a|`$ を抑えます。$`|x - a| < 1`$ に制限すると
$`|x| < |a| + 1`$ なので $`|x + a| \le |x| + |a| < 2|a| + 1`$。ゆえに

```math
|x^2 - a^2| = |x-a|\,|x+a| < (2|a| + 1)\,|x - a|.
```

$`\varepsilon > 0`$ に対し、$`\delta = \min\!\left(1,\ \dfrac{\varepsilon}{2|a| + 1}\right)`$
をとります。すると $`|x - a| < \delta`$ で
$`|x^2 - a^2| < (2|a|+1)\cdot \dfrac{\varepsilon}{2|a|+1} = \varepsilon`$。

**答え.** $`f`$ は $`a`$ で連続です。（コツは、まず $`\delta \le 1`$ で「余分な」因子
$`|x+a|`$ を抑え、それからさらに縮めて $`\varepsilon`$ に合わせること。積に対する標準的な
パターンです。）

## 有名な不連続性

$`x \neq 0`$ に対する関数 $`g(x) = \sin(1/x)`$ は、$`x \to 0`$ のとき **極限をもちません**。
$`x_n = \tfrac{1}{n\pi}`$ に沿うと $`g(x_n) = 0`$、$`x_n' = \tfrac{1}{2\pi n + \pi/2}`$ に
沿うと $`g(x_n') = 1`$。$`0`$ に近づく2つの数列が異なる極限を与えるので、数列による判定に
より極限は存在せず、$`g(0)`$ をどう定めてもそこで連続にはできません。

## よくある間違い

- **極限に $`x = a`$ を入れる.** 極限はわざと $`x = a`$ を無視します。$`f(a)`$ が未定義でも、
  別の値でも、$`\lim_{x\to a} f`$ は存在し得ます。
- **またしても量化子の順序.** $`\delta`$ は $`\varepsilon`$ と $`a`$ の両方に依存してよい。
  （1つの $`\delta`$ がすべての $`a`$ で同時に効くとき、$`f`$ は *一様* 連続 — 後のレッスンの
  より強い概念です。）
- **1つの数列で極限の存在を示す.** 1つの数列では *否定* しかできません。存在には
  $`\varepsilon`$–$`\delta`$ の条件全体（または全数列）が必要です。
- **「ペンを離さない」を証明と思う.** 絵的な発想は直感を導きますが論証ではありません。
  $`\varepsilon`$–$`\delta`$ か数列による判定に頼りましょう。

## 練習問題

1. 定義から、$`f(x) = 3x - 4`$ がすべての $`a`$ で連続であることを証明しなさい。
2. 数列による判定を用いて、$`\lim_{x\to 0}\dfrac{|x|}{x}`$ が存在しないことを示しなさい。
3. $`x \in \mathbb{Q}`$ のとき $`f(x) = x`$、$`x \notin \mathbb{Q}`$ のとき $`f(x) = 0`$ とする。
   $`f`$ はどの点で連続ですか。

<details>
<summary>解答</summary>

1. $`|f(x)-f(a)| = 3|x-a| < \varepsilon`$ は $`|x-a| < \tfrac\varepsilon3`$ のとき成立。
   $`\delta = \tfrac\varepsilon3`$ ととる（$`a`$ によらないので $`f`$ は一様連続でもある）。
2. $`x_n = \tfrac1n \to 0^+`$ に沿うと比は $`+1`$、$`x_n = -\tfrac1n \to 0^-`$ に沿うと
   $`-1`$。異なる極限 $`\Rightarrow`$ 極限なし。
3. $`a = 0`$ のみ。任意の $`x`$ で $`|f(x)| \le |x|`$ なので、$`x \to 0`$ のとき
   $`f(x) \to 0 = f(0)`$。$`a \neq 0`$ では、$`a`$ に近い有理数は $`a`$ に近い値を、無理数は
   $`0`$ を与え、両者が食い違うので $`f`$ はそこで不連続。

</details>
