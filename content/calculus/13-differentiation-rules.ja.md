# 微分の法則

## なぜ重要か

すべての導関数を極限の定義から計算するのは骨が折れますし、その必要もありません。導関数は
小さな **代数的法則** の集まり — 和・定数倍・積・商についての法則 — に従い、これにより
既知の関数のどんな組合せも機械的に微分できます。本レッスンでは4つの法則を述べ、それぞれを
定義から直接 *証明* します。暗記した公式ではなく、確かな土台の上に道具立てを置くためです。

以下を通して、$`f`$ と $`g`$ は $`a`$ で微分可能とし、$`c`$ を定数とします。

## 線形性：和と定数倍

> **定理（線形性）.** $`f + g`$ と $`cf`$ は $`a`$ で微分可能で、
> ```math
> (f + g)'(a) = f'(a) + g'(a), \qquad (cf)'(a) = c\,f'(a).
> ```

*証明.* 差分商を分けて極限に移します。
```math
\frac{(f+g)(a+h) - (f+g)(a)}{h}
= \frac{f(a+h) - f(a)}{h} + \frac{g(a+h) - g(a)}{h}
\to f'(a) + g'(a),
```
ここで[極限の演算](./08-functions-and-continuity.ja.md)を使いました。定数倍も同様です。
$`\dfrac{cf(a+h) - cf(a)}{h} = c\,\dfrac{f(a+h)-f(a)}{h} \to c\,f'(a)`$。$`\blacksquare`$

両者を合わせると、微分は *線形* な操作だといえます。$`(\alpha f + \beta g)' = \alpha f'
+ \beta g'`$。

## 積の法則

積の導関数は、導関数の積では **ありません**。正しい法則は、各関数を相手の導関数と
組み合わせます。

> **定理（積の法則）.** $`fg`$ は $`a`$ で微分可能で、
> ```math
> (fg)'(a) = f'(a)\,g(a) + f(a)\,g'(a).
> ```

*証明.* コツは、分子に $`f(a+h)\,g(a)`$ を足して引き、各部分を差分商にすることです。
```math
\begin{aligned}
\frac{f(a+h)g(a+h) - f(a)g(a)}{h}
&= \frac{f(a+h)\big[g(a+h) - g(a)\big] + g(a)\big[f(a+h) - f(a)\big]}{h} \\[2pt]
&= f(a+h)\,\frac{g(a+h) - g(a)}{h} + g(a)\,\frac{f(a+h) - f(a)}{h}.
\end{aligned}
```
ここで $`h \to 0`$ とします。2つの商は $`g'(a)`$ と $`f'(a)`$ に近づき、因子
$`f(a+h) \to f(a)`$ となります。なぜなら $`f`$ は微分可能なので
[$`a`$ で連続](./12-derivative-definition.ja.md)だからです。ゆえに極限は
$`f(a)g'(a) + g(a)f'(a)`$ です。$`\blacksquare`$

この呼び戻しが効いています — 積の法則は $`f(a+h) \to f(a)`$ を知るために「微分可能
$`\Rightarrow`$ 連続」を *必要* とします。

## 逆数の法則と商の法則

> **定理（逆数の法則）.** $`g(a) \neq 0`$ ならば $`1/g`$ は $`a`$ で微分可能で、
> ```math
> \left(\frac1g\right)'(a) = -\,\frac{g'(a)}{g(a)^2}.
> ```

*証明.* $`g`$ は連続で $`g(a) \neq 0`$ なので、$`g`$ は $`a`$ の近くで $`0`$ になりません。
よって次の商は小さな $`h`$ で定義されます。
```math
\frac1h\left(\frac{1}{g(a+h)} - \frac{1}{g(a)}\right)
= -\,\frac{g(a+h) - g(a)}{h}\cdot\frac{1}{g(a+h)\,g(a)}
\to -\,g'(a)\cdot\frac{1}{g(a)^2},
```
ここで $`g(a+h) \to g(a)`$ を使いました。$`\blacksquare`$

$`f/g = f \cdot (1/g)`$ と書いて積の法則と逆数の法則を合わせると、商の法則が得られます。

> **定理（商の法則）.** $`g(a) \neq 0`$ ならば $`f/g`$ は $`a`$ で微分可能で、
> ```math
> \left(\frac{f}{g}\right)'(a) = \frac{f'(a)\,g(a) - f(a)\,g'(a)}{g(a)^2}.
> ```

*証明.*
```math
\left(\frac{f}{g}\right)' = f'\cdot\frac1g + f\cdot\left(-\frac{g'}{g^2}\right)
= \frac{f'}{g} - \frac{f g'}{g^2} = \frac{f'g - f g'}{g^2}. \qquad \blacksquare
```
分子の順序が大切です — 「（分子の導関数）×（分母）*マイナス*（分子）×（分母の導関数）」
であって、逆ではありません。

## 正の整数べきの法則

> **系.** すべての整数 $`n \ge 1`$ に対し、$`\dfrac{d}{dx}\,x^n = n\,x^{n-1}`$。

*証明（$`n`$ についての帰納法）.* $`n = 1`$ では $`\dfrac{d}{dx}\,x = 1 = 1\cdot x^0`$。
$`\dfrac{d}{dx}\,x^n = n x^{n-1}`$ を仮定すると、$`x^{n+1} = x \cdot x^n`$ に積の法則を
適用して
```math
\frac{d}{dx}\,x^{n+1} = 1\cdot x^n + x\cdot n x^{n-1} = x^n + n x^n = (n+1)\,x^n.
\qquad \blacksquare
```
線形性と合わせれば、これで **あらゆる多項式** を項ごとに微分できます。（この法則は実は
すべての実数の指数で成り立ちます。一般の場合は指数関数が使えるようになってから証明
します。）

## 例題

**例1 — 多項式.** $`p(x) = 3x^4 - 5x^2 + 2x - 7`$ に対し、線形性とべきの法則から
```math
p'(x) = 3\cdot 4x^3 - 5\cdot 2x + 2 = 12x^3 - 10x + 2.
```
（定数 $`-7`$ の寄与は $`0`$。）

**例2 — 商.** $`f(x) = \dfrac{x^2 - 1}{x^2 + 1}`$ に対し、分子 $`u = x^2-1`$、分母
$`v = x^2+1`$ で商の法則を使うと
```math
f'(x) = \frac{(2x)(x^2+1) - (x^2-1)(2x)}{(x^2+1)^2}
= \frac{2x\big[(x^2+1) - (x^2-1)\big]}{(x^2+1)^2}
= \frac{4x}{(x^2+1)^2}.
```

## よくある間違い

- **$`(fg)' = f'g'`$.** 誤り。$`f'g + fg'`$ を使います。交差項が本質的です。
- **商の法則で符号や順序を誤る.** $`g^2`$ の上は $`f'g - fg'`$ であって $`fg' - f'g`$ では
  ありません。順序を入れ替えると、すべての答えの符号が反転します。
- **$`g^2`$ を落とす.** 商の法則と逆数の法則は、分母の *2乗* で割ります。
- **定数の導関数を $`0`$ 以外にする.** 定数は変化率が $`0`$ です。$`c' = 0`$。
- **連続性が効いていることを忘れる.** 積と逆数の証明は $`f(a+h) \to f(a)`$ と
  $`g(a+h) \to g(a)`$ — すなわち「微分可能 $`\Rightarrow`$ 連続」 — に依拠します。

## 練習問題

1. $`(2x^3 - x)(x^2 + 4)`$ を2通り — 積の法則と、先に展開する方法 — で微分し、
   答えが一致することを確かめなさい。
2. $`\dfrac{x}{x^2 + 1}`$ を微分しなさい。
3. 逆数の法則で $`\dfrac{1}{x^2 + 1}`$ を微分し、商の法則でも確かめなさい。
4. 商の法則の分母のべきを導きなさい — なぜ $`g`$ ではなく $`g^2`$ が現れるのか説明しなさい。

<details>
<summary>解答</summary>

1. 積の法則：$`(6x^2 - 1)(x^2+4) + (2x^3 - x)(2x) = 6x^4 + 24x^2 - x^2 - 4 + 4x^4 -
   2x^2 = 10x^4 + 21x^2 - 4`$。先に展開：$`2x^5 + 8x^3 - x^3 - 4x = 2x^5 + 7x^3 - 4x`$、
   その導関数は $`10x^4 + 21x^2 - 4`$。一致します。
2. $`\dfrac{(1)(x^2+1) - x(2x)}{(x^2+1)^2} = \dfrac{1 - x^2}{(x^2+1)^2}`$。
3. 逆数：$`-\dfrac{2x}{(x^2+1)^2}`$。分子 $`1`$（導関数 $`0`$）の商の法則：
   $`\dfrac{0\cdot(x^2+1) - 1\cdot 2x}{(x^2+1)^2} = -\dfrac{2x}{(x^2+1)^2}`$。同じです。
4. $`f/g = f\cdot(1/g)`$ と $`(1/g)' = -g'/g^2`$ から、$`g^2`$ は逆数の法則由来であり、
   そこでは $`\tfrac{1}{g(a+h)\,g(a)} \to \tfrac1{g^2}`$ として現れます。

</details>
