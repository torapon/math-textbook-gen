# 有界な単調数列は収束する

## なぜ重要か

$`\varepsilon`$–$`N`$ の定義は極限を *確かめる* 方法を教えてくれますが、候補の値 $`L`$ が
すでに手元にあることを前提にしています。ところが、しばしばそれがありません。ある
過程の終着点（小数展開、反復計算、部分和の数列）として数を作り、極限の名前を言える
**前に** それが収束することを知りたいのです。単調収束定理は、極限の存在を文字どおり
何もないところから生み出す最初の道具であり、用いるのは実数の完備性だけです。

## 定義

> **定義（単調）.** 数列 $`(a_n)`$ が、すべての $`n`$ で $`a_n \le a_{n+1}`$ を満たすとき
> *増加*、$`a_n \ge a_{n+1}`$ を満たすとき *減少* といいます。どちらかであるとき
> *単調* といいます。（不等号を狭義にすると *狭義* 増加／減少です。）

## 主な結果

> **定理（単調収束）.** 単調数列が収束する **必要十分条件** は、有界であることである。
> より詳しくは、
> ```math
> (a_n) \text{ が増加かつ上に有界} \implies \lim_{n\to\infty} a_n = \sup_n a_n,
> ```
> ```math
> (a_n) \text{ が減少かつ下に有界} \implies \lim_{n\to\infty} a_n = \inf_n a_n.
> ```

*証明（増加の場合）.* $`(a_n)`$ が増加かつ上に有界とします。集合
$`S = \{a_n : n \ge 1\}`$ は空でなく上に有界なので、[完備性定理](./02-supremum-infimum.ja.md)
により上限 $`\alpha = \sup S`$ をもちます。$`a_n \to \alpha`$ を示します。

$`\varepsilon > 0`$ とします。上限の特徴づけにより $`\alpha - \varepsilon`$ は上界では
**ない** ので、ある項が $`a_N > \alpha - \varepsilon`$ を満たします。数列は増加なので、
$`n \ge N`$ なるすべての $`n`$ で、

```math
\alpha - \varepsilon < a_N \le a_n \le \alpha < \alpha + \varepsilon
```

となり、$`|a_n - \alpha| < \varepsilon`$ です。これがまさに $`a_n \to \alpha`$ です。減少の
場合は $`(-a_n)`$ に適用すれば従います。（$`\Leftarrow`$ がこの定理で、$`\Rightarrow`$ は
収束数列が有界であることから成り立ちます。）$`\blacksquare`$

イメージ：増加数列は登り続けますが、天井の下に押さえられています。最も低い天井 —
すなわち上限 — こそが、その数列が近づかざるを得ない値です。単調性により上に
飛び越えることはなく、$`\alpha`$ より下で止まることもありません（止まればその低い高さが
より小さい天井になってしまうからです）。

## 例題

**問題.** $`a_1 = \sqrt{2}`$, $`a_{n+1} = \sqrt{2 + a_n}`$ で定める。$`(a_n)`$ が収束する
ことを示し、その極限を求めなさい。

*上に $`2`$ で有界.* 帰納法による：$`a_1 = \sqrt2 < 2`$、また $`a_n < 2`$ なら
$`a_{n+1} = \sqrt{2 + a_n} < \sqrt{2+2} = 2`$。

*増加.* $`a_{n+1} \ge a_n \iff 2 + a_n \ge a_n^2 \iff a_n^2 - a_n - 2 \le 0
\iff (a_n - 2)(a_n + 1) \le 0`$ で、$`0 < a_n < 2`$ より成立します。

よって $`(a_n)`$ は増加かつ上に有界で、定理よりある $`L`$ に収束します。
$`a_{n+1} = \sqrt{2 + a_n}`$ で $`n \to \infty`$ とすると（両辺の極限は等しい）
$`L = \sqrt{2 + L}`$、すなわち $`L^2 - L - 2 = 0`$ なので $`(L-2)(L+1)=0`$。
$`L \ge a_1 > 0`$ より $`L = 2`$ をとります。

**答え.** $`a_n \to 2`$。

2段階のパターンに注目してください。**(1)** 単調＋有界で収束を示し、**(2)** *その後で*
漸化式を使って極限を求めます。手順2だけでは不正です。方程式 $`L = \sqrt{2+L}`$ は根
$`-1`$ ももち、手順1なしでは発散の可能性を排除できません。

## よくある間違い

- **収束を示す前に不動点方程式を解く。** 発散数列には代入すべき極限がなく、その計算は
  無意味です。
- **単調性を当然と思い込む。** 単調性は（多くは帰納法で）証明すべきもので、数項を
  眺めて判断してはいけません。
- **有界 *または* 単調のどちらか一方だけ。** 有界だが単調でない：$`(-1)^n`$（発散）。
  単調だが有界でない：$`a_n = n`$（$`+\infty`$ に発散）。両方が必要です。

## 練習問題

1. $`a_1 = 1`$, $`a_{n+1} = \tfrac12\!\left(a_n + \tfrac{2}{a_n}\right)`$ とする。
   $`(a_n)`$ が最初の一歩のあと $`n \ge 2`$ で減少し、$`\sqrt2`$ で下に有界であることを
   示し、極限を求めなさい。
2. 数列 $`b_n = \left(1 + \tfrac1n\right)^n`$ が増加かつ $`3`$ で上に有界であり、ゆえに
   収束することを示しなさい。*（その極限は数 $`e`$ です。）*
3. 収束 **しない** 増加数列を1つ挙げ、どの仮定が成り立たないかを説明しなさい。

<details>
<summary>解答</summary>

1. （$`\sqrt2`$ に対するニュートン反復。）$`a_n > 0`$ のとき相加相乗平均により
   $`a_{n+1} = \tfrac12(a_n + \tfrac2{a_n}) \ge \sqrt{a_n \cdot \tfrac2{a_n}} = \sqrt2`$
   なので $`n \ge 2`$ で $`a_n \ge \sqrt2`$。さらに
   $`a_{n+1} - a_n = \tfrac12(\tfrac2{a_n} - a_n) = \tfrac{2 - a_n^2}{2a_n} \le 0`$ で
   減少かつ下に有界。極限 $`L`$ は $`L = \tfrac12(L+\tfrac2L)`$ を満たし $`L = \sqrt2`$。
2. 標準的。二項定理で展開して項ごとに比べると単調性が、和を等比級数で抑えると
   $`< 3`$ の有界性が出ます。単調＋有界 $`\Rightarrow`$ 収束（極限は $`e`$）。
3. $`a_n = n`$：増加だが上に有界でないので、有界性の仮定が破れ $`+\infty`$ に発散します。

</details>
